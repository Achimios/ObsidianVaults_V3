# Lp Norm 3D World — 人类解释文档

## 这个项目是什么？

从豆包的 L3 范数可视化出发，逐步扩展为一个可探索的 3D 数学世界。
核心概念：**Lp 范数**定义了空间中"球"的形状。当 p 变化时，球的形状从尖锐八面体到圆球再到立方体。

$$\|x\|_p = \left(\sum_i |x_i|^p\right)^{1/p}$$

| p 值 | 形状名称 | 视觉外观 |
|---|---|---|
| p = 1 | L¹ 球（八面体） | 尖锐菱形 |
| p = 2 | L² 球（欧氏球） | 完美圆球 |
| p = 3 | L³ 球 | 介于球和方块之间 |
| p → ∞ | L∞ 球（超立方体） | 正方体 |

## 代码结构

```
Copilot的代码/src/
├── main.py              ← 入口，运行这个
├── world_objects.py     ← 场景对象（make_lp_sphere、地板等）
├── camera_controller.py ← FPS + 上帝相机（FPSGLView 类）
└── ui_panel.py          ← 右侧参数面板

来自豆包/src/
├── 0_L3范数空间-3维-球体.py       ← 最基础版本
├── 1_L3范数3D小游戏.py            ← 豆包第一版（本项目起点）
├── 2_L3范数3D小游戏_真L3视角.py  ← 真 L3 空间的摄像机旋转
└── 3_L3范数3D小游戏_极坐标.py    ← 极坐标表示
```

## 操作说明（V2 Copilot 版）

| 按键 | 功能 |
|---|---|
| **Tab** | 切换 上帝视角 / FPS 视角 |
| **WASD** | FPS 模式下前后左右移动 |
| **Space** | FPS 模式上升 |
| **Ctrl** | FPS 模式下降 |
| **右键拖拽** | FPS 模式转头（视角旋转） |
| **左键拖拽** | 上帝模式轨道旋转 |
| **滚轮** | 上帝模式缩放 |

右侧面板：
- **p 值滑块**：实时改变 5 个可调小球的 Lp 形状（0.7 ~ 8.0）
- **缩放距离**：上帝模式下的视距
- **移动速度**：FPS 模式移动速度
