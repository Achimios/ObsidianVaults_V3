# -CONTEXT — L1 vs L2 损失函数 PID 整定 NN 实验

## 这是什么
验证实验：对比 L1（MAE）损失 vs L2（MSE）损失在浅层 NN 做 PID 整定时的差异。
重点观察：加入异常样本（传感器故障模拟）时，两种损失函数的抗噪能力对比。

起因：`0_Sonnet的思考.md` 操作 2 — 验证"L1 对异常值鲁棒"的实际效果。

## 设计要点（待实现时参考）
- 被控对象：仿真一阶惯性系统 $G(s) = k / (\tau s + 1)$，不同 τ 对应不同最优 Kp
- 网络结构：1 隐层，5~10 神经元，输入维度 3（误差 e、误差变化率、工况参数 τ）
- 对比维度：
  1. 正常数据：L1 vs L2 收敛速度
  2. 含 5% 异常样本时：L1 vs L2 拟合误差
  3. 工况外推时：L1 vs L2 泛化能力
- 关键观察图：散点图（预测 Kp vs 真实 Kp），分 L1/L2/Huber 三组

## 跨 session 提醒
- **L1 正则化 ≠ MAE 损失 ≠ L1 度量空间**（三个不同概念，代码里要分开测）
- 豆包"90% 权重归零"是夸大说法——L1 正则化诱导稀疏，但要强到这种程度会严重损精度
- Huber 损失是 L1 和 L2 的平滑过渡，实用性最强
