# 项目架构 — 滤波交互式分析仪

```xml
<project name="01_滤波交互式分析仪">

  <entry>main.py</entry>
  <!-- FilterAnalyzer 主类 + main()。__init__ 在这里，状态变量在这里。 -->

  <layer name="constants">constants.py</layer>
  <!-- 采样率 FS=2000, N_SECONDS=30, N_SIG=60000 + DARK/LIGHT 主题字典 -->

  <layer name="dsp">dsp.py</layer>
  <!-- 纯 numpy/scipy 数学函数，无 Qt 依赖 -->
  <!-- perlin_noise_1d / pt1_coeffs / lkf_coeffs / resonance / notch_coeffs / resonance_dist -->

  <layer name="ui">ui_mixin.py</layer>
  <!-- class UIMixin: 左侧参数面板所有小控件 + _build_ui() -->

  <layer name="interact">interact_mixin.py</layer>
  <!-- class InteractMixin: 打杆曲线 — 新增/删除/调整+锚点 -->

  <layer name="draw">draw_mixin.py</layer>
  <!-- class DrawMixin: _do_update() matplotlib 5图绘制 + _toggle_x/y + _sync_lkf_to_pt1 -->

  <layer name="theme">theme_mixin.py</layer>
  <!-- class ThemeMixin: Mirror Edge 双主题切换 + toolbar 图标反色 -->

</project>
```

## 继承关系

```
FilterAnalyzer(ThemeMixin, InteractMixin, DrawMixin, UIMixin, QMainWindow)
      │
      ├─ ThemeMixin   → _toggle_theme, _update_toolbar_icons
      ├─ InteractMixin→ canvas 鼠标事件
      ├─ DrawMixin    → _do_update (主绘图循环)
      ├─ UIMixin      → _build_ui (左面板)
      └─ QMainWindow  → Qt 窗口基础
```
