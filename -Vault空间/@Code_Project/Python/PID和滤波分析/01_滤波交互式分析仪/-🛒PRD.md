# -🛒PRD — 滤波交互式分析仪

## 产品目标
在交互式 GUI 中直观分析不同滤波器对无人机陀螺仪采样数据的处理效果，
辅助调参：在幅频、相频、群延迟、PSD、时域五个维度同步观察各滤波配置的差异。

## 核心使用场景
1. **滤波器参数调优**：实时调整 PT1 截止频率、LKF 噪声参数，对比响应速度 vs 噪声抑制能力
2. **Notch 滤波设计**：定位机架共振峰（PSD 图），放置 Notch A/B 精准压制
3. **打杆信号仿真**：注入自定义打杆曲线（手动 Cubic 或正弦注入），观察滤波器对快速指令的响应延迟
4. **多滤波器对比**：在同一画布中对比 PT1、LKF、PT1+Notch、LKF+Notch 四条曲线

## 输入数据
- 当前：程序内部生成合成陀螺噪声（Perlin + 白噪声 + 机架共振模拟），`fs = 2000 Hz`
- 规划（Phase 2）：加载真实飞控 `.csv` / Blackbox 日志文件的陀螺通道

## 技术约束
- **采样率**：固定 `FS = 2000 Hz`（与 Betaflight 典型陀螺采样一致）
- **时域长度**：30 秒，60000 个样本点
- **单位**：角速度 dps，增益 × 线性倍数（不用 dB），延迟 ms
- **运行环境**：Windows 系统级 Python，`cd src && py main.py`
