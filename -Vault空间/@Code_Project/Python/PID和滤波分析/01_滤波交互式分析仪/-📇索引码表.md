# -📇 注释码表 — 索引码表
> **用途**：登记所有 `>v<` 特殊标注字符串，方便全项目搜索和 AI 快速定位关键点。


## 📌 本项目专属规则
1. **遇到 `>v<` 注释**：先查 `-📇索引码表.md`，确认含义后再修改
2. **新增 `>v<` 标注前**：在码表中登记（否则维护脚本会报"未登记"）
3. **推荐库**：这里可以写如 "本项目角度环 AHRS 算法用 `Mahony`，禁止替换成 `EKF3`"(这是给位置和速度环用的！)


## 格式规则
```
>v<  + 意义EMOJI + 主标题 - 副标题 - 额外注释
```

- `>v<` = 标注起始符
- 码表只记录到**主标题**，副标题留在代码注释里
- 搜索：`grep -r ">v<" ./` 即可定位所有标注
