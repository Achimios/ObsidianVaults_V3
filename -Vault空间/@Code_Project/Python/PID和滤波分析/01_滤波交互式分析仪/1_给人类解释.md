## 什么是信号

信号是**随时间变化的量**。陀螺仪输出的角速度 $\omega(t)$ 是信号。你心脏的心电图是信号。股票价格也是信号。

信号分两种：
- **连续信号** $x(t)$：时间是连续的实数。真实物理世界的信号都是连续的。
- **离散信号** $x[n]$：每隔 $T_s$ 秒采一个样。$n$ 是整数。计算机只能处理离散信号。

> **采样频率** $f_s = 1/T_s$。本分析仪 $f_s = 2000\text{Hz}$，即每秒采 2000 个点。

### 频率的含义

任何信号都可以分解为不同频率的正弦波之和（傅里叶定理）。

- **低频**：信号变化慢 → 你真正关心的"趋势"
- **高频**：信号变化快 → 通常是噪声、振动
- **Nyquist 频率** $f_N = f_s/2$：离散系统能表达的最高频率。$f_s=2000$ 时 $f_N=1000\text{Hz}$。

## 什么是滤波器

滤波器是一个**输入→输出**的系统：给它一个信号 $x$，它吐出经过处理的信号 $y$。

```
x[n] ──→ [ 滤波器 ] ──→ y[n]
```

**低通滤波器**：让低频通过，衰减高频。本质是"平滑"。
**Notch(陷波)滤波器**：在某个特定频率处挖一个深坑，其他频率不动。
**高通滤波器**：让高频通过，衰减低频（本分析仪不涉及，但概念对称）。

### 线性时不变（LTI）

本分析仪中所有滤波器都是 **LTI** 系统：
- **线性**：$F(ax + by) = aF(x) + bF(y)$。放大输入 → 输出等比放大。
- **时不变**：延迟输入 → 输出等量延迟。系统特性不随时间变化。

LTI 系统的全部行为由**传递函数**唯一确定。

## 连续世界与离散世界 — s域和z域

### 拉普拉斯变换（s域）

连续时间信号/系统的"频率肖像"工具。

$$X(s) = \int_0^\infty x(t) e^{-st} dt$$

其中 $s = \sigma + j\omega$ 是复变量。当 $\sigma = 0$ 时，$s = j\omega$ 直接对应频率，此时 $X(j\omega)$ 就是信号的频谱。

**直觉**：拉普拉斯变换把"随时间变化"翻译成"在频率上的权重分布"。微分变成乘 $s$，积分变成除 $s$。这让微分方程变成代数方程。

### Z变换（z域）

离散时间信号/系统的"频率肖像"工具。

$$X(z) = \sum_{n=0}^{\infty} x[n] z^{-n}$$

其中 $z = e^{sT_s} = e^{j\omega T_s}$（在单位圆上）。$z^{-1}$ 表示"延迟一个采样周期"。

| | 连续域 | 离散域 |
|---|---|---|
| 变量 | $s$ | $z$ |
| 延迟 | $e^{-sT}$ | $z^{-1}$ |
| 微分 | $\times s$ | $\times (1-z^{-1})/T_s$（前向差分近似）|
| 稳定区域 | $\text{Re}(s) < 0$（左半平面）| $|z| < 1$（单位圆内）|
| 频率对应 | $s = j\omega$ | $z = e^{j\omega T_s}$（单位圆）|

## 传递函数 H(s) — 给系统拍一张频率肖像

传递函数定义：$H(s) = \frac{Y(s)}{X(s)}$，即输出与输入的比值（在频域中）。

### 例：RC 低通电路

物理世界最简单的低通：电阻 $R$ + 电容 $C$。

$$H(s) = \frac{1}{1 + sRC} = \frac{\omega_c}{s + \omega_c}$$

其中 $\omega_c = 1/(RC) = 2\pi f_c$ 是截止角频率。

**它在说什么**：
- 低频（$s \to 0$）：$H \to 1$，信号完整通过
- 截止频率处（$s = j\omega_c$）：$|H| = 1/\sqrt{2} \approx 0.707$（-3dB）
- 高频（$s \to \infty$）：$H \to 0$，信号被衰减

### z域传递函数

$$H(z) = \frac{B(z)}{A(z)} = \frac{b_0 + b_1 z^{-1} + b_2 z^{-2} + \cdots}{1 + a_1 z^{-1} + a_2 z^{-2} + \cdots}$$

这个分式直接决定滤波器的一切行为。b 和 a 系数就是滤波器的"DNA"。

## 从连续到离散 — 离散化方法

物理世界是连续的，计算机是离散的。**离散化** = 把连续系统 H(s) 变成离散系统 H(z)。

### 前向欧拉法（Euler, RK1）

最简单的近似：用当前值估算下一步。

$$s \approx \frac{z - 1}{T_s} \quad \Leftrightarrow \quad z = 1 + sT_s$$

**直觉**：微分 $dx/dt$ 用前向差分 $(x[n+1] - x[n])/T_s$ 替代。

**优点**：计算快，Betaflight 飞控用的就是这个。
**缺点**：

- **精度低**：只是一阶近似。真实的 -3dB 频率会偏离设计值（尤其当 $f_c$ 接近 $f_s/2$ 时严重漂移）。
- **可能不稳定**：如果步长 $T_s$ 太大，系统会发散。

> **本分析仪中**：PT1 Euler 模式 = Betaflight 实际飞的算法。设定 100Hz，实际 -3dB 在 ~97Hz 左右。

### 双线性变换（Bilinear / Tustin）

更精确的近似：用梯形积分替代欧拉积分。

$$s = \frac{2}{T_s} \cdot \frac{z - 1}{z + 1} \quad \Leftrightarrow \quad z = \frac{1 + sT_s/2}{1 - sT_s/2}$$

**直觉**：把 s 平面的虚轴**精确映射**到 z 平面的单位圆。这意味着：

- 连续系统的稳定性 → 离散系统也稳定（保证）
- 频率会发生"非线性压缩"（频率翘曲），但有精确的反公式

**频率翘曲**：$\omega_d = \frac{2}{T_s} \tan\left(\frac{\omega_a T_s}{2}\right)$

其中 $\omega_a$ 是模拟频率，$\omega_d$ 是数字频率。低频时几乎无失真，高频时压缩严重。

**优点**：精度高，稳定性有保证，工业标准。
**缺点**：高频翘曲（可以预补偿）。

> **本分析仪中**：PT1 Bilinear/Tustin 模式。设定 100Hz，实际 -3dB 精确在 100Hz。

### RK4（四阶龙格-库塔）

不是用于传递函数离散化，而是用于**ODE（常微分方程）数值积分**。

每一步用 4 次函数求值来估算斜率：

$$k_1 = f(t_n, y_n)$$
$$k_2 = f(t_n + h/2, y_n + hk_1/2)$$
$$k_3 = f(t_n + h/2, y_n + hk_2/2)$$
$$k_4 = f(t_n + h, y_n + hk_3)$$
$$y_{n+1} = y_n + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4)$$

| 方法 | 阶数 | 精度 | 每步计算量 | 用途 |
|---|---|---|---|---|
| Euler (RK1) | 1 | $O(h)$ | 1 次 | 飞控实时（追求速度）|
| RK4 | 4 | $O(h^4)$ | 4 次 | 仿真（追求精度）|
| Bilinear | — | 频域精确映射 | 1 次 | 传递函数离散化 |

**Euler vs Bilinear vs RK4 的本质区别**：
- Euler 和 Bilinear 是**整个传递函数**的离散化（把 H(s) 变成 H(z)）
- RK4 是**微分方程**的数值求解（把 $\dot{x} = f(x,t)$ 一步步积分）

## 差分方程 DEQ — 滤波器的"执行指令"

H(z) 是**频率肖像**（描述滤波器做什么），DEQ 是**时域指令**（描述滤波器怎么算）。

### 从 H(z) 到 DEQ

$$H(z) = \frac{Y(z)}{X(z)} = \frac{b_0 + b_1 z^{-1} + b_2 z^{-2}}{1 + a_1 z^{-1} + a_2 z^{-2}}$$

交叉相乘：$A(z)Y(z) = B(z)X(z)$

逆 Z 变换（$z^{-k}$ → 延迟 k 步）：

$$y[n] + a_1 y[n{-}1] + a_2 y[n{-}2] = b_0 x[n] + b_1 x[n{-}1] + b_2 x[n{-}2]$$

整理：

$$\boxed{y[n] = b_0 x[n] + b_1 x[n{-}1] + b_2 x[n{-}2] - a_1 y[n{-}1] - a_2 y[n{-}2]}$$

这就是 DEQ。就是一行代码：

```python
y[n] = b[0]*x[n] + b[1]*x[n-1] + b[2]*x[n-2] - a[1]*y[n-1] - a[2]*y[n-2]
```

**H(z) 和 DEQ 用完全相同的 b, a 系数**。只是写法不同：一个在频域（分式），一个在时域（递推）。

### IIR vs FIR

- **IIR（无限脉冲响应）**：有 $a$ 系数（反馈 $y[n-k]$）。用自己的历史输出来计算。理论上脉冲响应无限长。PT1、LKF、Notch 都是 IIR。
- **FIR（有限脉冲响应）**：只有 $b$ 系数，$a = [1]$。只用输入的历史。脉冲响应有限长。

IIR 用少量系数就能做到很陡的衰减，但可能振荡。FIR 更稳定但需要更多系数。

## PT1 — 最简单的低通滤波器

PT1 = "一阶惯性环节"（德语行业术语：Proportional-Transfer 1st order）。

### 连续域

$$H(s) = \frac{\omega_c}{s + \omega_c} = \frac{1}{1 + s/\omega_c}$$

只有一个参数：截止频率 $f_c$（$\omega_c = 2\pi f_c$）。

### 离散化后

**Euler**：

$$y[n] = \alpha \cdot x[n] + (1 - \alpha) \cdot y[n{-}1], \quad \alpha = \frac{\omega_c T_s}{1 + \omega_c T_s} \approx \omega_c T_s$$

这就是加权移动平均！$\alpha$ 越小越平滑，越大越跟踪。

**Bilinear**：

$$b = [b_0, b_0], \quad a = [1, a_1]$$
$$b_0 = \frac{\omega_c T_s / 2}{1 + \omega_c T_s / 2}, \quad a_1 = \frac{\omega_c T_s / 2 - 1}{1 + \omega_c T_s / 2}$$

注意 Bilinear 的 $b_0 = b_1$（分子对称），这是梯形积分的特征。

### 物理类比

PT1 就是一个弹簧阻尼器：给它一个阶跃输入，它指数趋近目标值。时间常数 $\tau = 1/\omega_c$。

温度计就是一个 PT1：环境温度突变，它慢慢跟上去。

## Kalman 滤波器 LKF — 最优估计器

Kalman 滤波器不是"设计出来的滤波器"，是**从概率论推导出的最优估计算法**。

### 问题设定

你有一个**状态**（不可直接观测），和一个**测量**（有噪声的观测）。

本分析仪的 LKF：

- **状态**：$\mathbf{x} = [\omega, \text{bias}]^T$（真实角速度 + 陀螺偏差）
- **状态转移**：$\omega_{n+1} = \omega_n - \text{bias}_n$，$\text{bias}_{n+1} = \text{bias}_n$
- **测量**：$y_n = H \cdot \mathbf{x}_n + v_n$（测量值 = 真实值 + 噪声 $v$）

### 三个调谐参数的物理含义

| 参数 | 含义 | 直觉 |
|---|---|---|
| $q_\omega$ | 角速度的**过程噪声方差** | "我相不相信角速度在变？" 越大 → 越信测量 → 越不平滑 |
| $q_b$ | bias 的**过程噪声方差** | "我相不相信 bias 在漂？" 越大 → bias 跟踪越灵敏 |
| $r$ | 测量噪声方差 | "测量有多吵？" 越大 → 越不信测量 → 越平滑 |

**截止频率由 $q_\omega / r$ 比值决定**（当 $q_b \approx 0$ 时）。比值大 → 高截止 → 少平滑。比值小 → 低截止 → 强平滑。

### LKF 运行时 = 线性滤波器

Kalman 滤波器在**设计阶段**需要迭代求解 Riccati 方程（非线性）。但一旦收敛，得到的 Kalman 增益 $K$ 是**固定常数**。此后 LKF = 线性差分方程 = 普通 IIR 滤波器。

这就是为什么本分析仪可以把 LKF 和 PT1 放在同一个 Bode 图上比较——运行时它们都是 b, a 系数的线性系统。

### H=[1,1] vs H=[1,0]

- **H=[1,1]**：测量 = $\omega + \text{bias}$。滤波器同时估计 $\omega$ 和 bias。这产生 2 阶系统，可能有**谐振峰**（低频增益突起 > 1）。
- **H=[1,0]**：测量 = $\omega$。不观测 bias。等价于 1 阶低通（无谐振），行为类似 PT1 但参数化方式不同。

### DC 增益恒等于 1.0

Kalman 滤波器是**无偏估计器**。数学证明：

$$\text{DC gain} = C(I - A')^{-1}K = 1 \quad \text{(恒等式)}$$

无论参数如何，在 DC（完全不变的信号）处，估计值 = 测量值。

## Notch — 精确狙杀单一频率

Notch 滤波器在中心频率 $f_0$ 处创造一个幅度"深坑"，其他频率基本不变。

### 传递函数

$$H(z) = \frac{1 - 2\cos(\omega_0 T_s)z^{-1} + z^{-2}}{1 - 2r\cos(\omega_0 T_s)z^{-1} + r^2 z^{-2}}$$

$Q$ 值控制坑的宽度：Q 越大坑越窄。$r$ 越接近 1 坑越深。

### 物理类比

想象一个共振弹簧。Notch 就是在弹簧共振频率处加一个反相阻尼器，精确抵消那个频率的振动。

在无人机中：桨叶以固定频率振动（一般 100-300Hz），Notch 精确消除这个频率，避免影响飞控。

## 频率响应 — Bode图在说什么

Bode 图是滤波器的"全频体检报告"。

### 幅频图（Magnitude）

$|H(e^{j\omega})|$ vs 频率。告诉你每个频率被放大还是衰减了多少。

- $|H| = 1$（0 dB）：信号原样通过
- $|H| = 0.707$（-3 dB）：**-3dB 点** = 截止频率的定义
- $|H| = 0$（-∞ dB）：信号被完全消灭

### 相频图（Phase）

$\angle H(e^{j\omega})$ vs 频率。告诉你每个频率的信号被延迟了多少相位。

- 0°：不延迟
- -90°：延迟四分之一个周期
- -180°：延迟半个周期（信号反相）

**重要**：相位延迟在闭环控制中至关重要。如果反馈信号延迟太多（相位裕度不足），系统会振荡甚至失控。

### 群延迟（Group Delay）

$$\tau_g(\omega) = -\frac{d\phi(\omega)}{d\omega}$$

群延迟是相位对频率的导数的负值，单位是**采样数**（或时间）。

**直觉**：如果你发射一个窄带信号包络，群延迟告诉你这个包络被延迟了多久。

- 常数群延迟 = 所有频率等量延迟 = 波形不变形（线性相位 FIR 的特征）
- 群延迟不均匀 = 不同频率延迟不同 = 波形畸变

### 1000Hz 处永远 -180°？

在本分析仪中，LKF 在 Nyquist 频率（1000Hz）处总是 -180°。这不是 bug，是**数学性质**：

LKF 的 $b[0] = 0$，意味着 $H(z) = z^{-1} \cdot G(z)$。在 Nyquist 处 $z = e^{j\pi} = -1$：

$$z^{-1} = -1 \quad \Rightarrow \quad \angle z^{-1} = -180°$$

$G(-1)$ 对低通滤波器是正实数（相位 0°），所以总相位 = $0° + (-180°) = -180°$。

## 功率谱 PSD 与 ASD — 噪声的频率指纹

### 功率谱密度 PSD

$$S_{xx}(f) \quad [\text{单位}^2/\text{Hz}]$$

PSD 告诉你信号的**功率**如何分布在各个频率上。对陀螺仪：$[\text{dps}^2/\text{Hz}]$。

**计算方法**：Welch 法——把长信号分段，每段做 FFT 取模方再平均。减少估计方差。

### 幅度谱密度 ASD

$$\sqrt{S_{xx}(f)} \quad [\text{单位}/\sqrt{\text{Hz}}]$$

ASD = PSD 开根号。对陀螺仪：$[\text{dps}/\sqrt{\text{Hz}}]$。数值更直观——它的积分近似等于信号的 RMS。

### 为什么 PSD 而不是直接 FFT？

FFT 的结果随信号长度变化（更长的信号 → 更大的 FFT 值）。PSD **归一化到每 Hz 的功率**，不依赖信号长度。这才是可比较、可重复的"噪声指纹"。

## 闭环控制 — PID 的本质

### 开环 vs 闭环

- **开环**：我拧到 50% 油门 → 飞机可能稳也可能翻。没有自动修正。
- **闭环（反馈控制）**：我设定目标角速度 → 传感器测量实际角速度 → 计算误差 → 调整输出 → 循环。

```
Setpoint ──→ [+] ──→ [PID] ──→ [电机] ──→ [机体] ──→ [陀螺] ──→
              ↑ (-)                                              |
              └──────────────────────────────────────────────────┘
```

### PID 三项的物理直觉

**误差** $e[n] = \text{setpoint}[n] - \text{measurement}[n]$

| 项 | 公式 | 直觉 | 如果只有这一项 |
|---|---|---|---|
| **P**（比例）| $K_p \cdot e$ | "当前差多少就纠多少" | 有稳态误差（永远差一点） |
| **I**（积分）| $K_i \sum e$ | "积累的历史误差也要补偿" | 消除稳态误差，但可能振荡 |
| **D**（微分）| $K_d \cdot \frac{de}{dt}$ | "误差在变大还是变小？提前刹车" | 抑制振荡，但放大噪声 |

### 传递函数视角

PID 控制器本身也是一个传递函数：

$$C(s) = K_p + \frac{K_i}{s} + K_d s = \frac{K_d s^2 + K_p s + K_i}{s}$$

**D 项的问题**：纯微分 $K_d s$ 在高频增益无限大 → 放大噪声。解决方案：D-LP 低通滤波器。

$$C_D(s) = \frac{K_d s}{1 + s/(2\pi f_{D\text{-LP}})}$$

### 闭环传递函数

设控制器 $C(s)$，被控对象（电机+机体）$P(s)$，滤波器 $F(s)$：

$$T(s) = \frac{C(s) \cdot P(s)}{1 + C(s) \cdot P(s) \cdot F(s)}$$

- 分子 = "开环增益"
- 分母的 $1 + L(s)$ 是关键：$L(s) = C \cdot P \cdot F$ 是**环路增益**。
- 如果 $|L(j\omega)| = 1$ 且 $\angle L(j\omega) = -180°$，分母 = 0 → 系统振荡 → **不稳定**！

这就是为什么滤波器的**相位**如此重要：它直接影响系统的相位裕度和稳定性。

## 为什么近似？工程视角

### 为什么飞控不用 Bilinear？

Betaflight 的 PT1 用 Euler 而不是 Bilinear，原因：

1. **计算量**：Euler = 1 次乘法 + 1 次加法。Bilinear = 2 次乘法 + 2 次加法（多一倍）。在 32kHz 循环率、几十个滤波器级联时，这很重要。
2. **精度损失可接受**：$f_c = 100$Hz, $f_s = 2000$Hz 时，Euler 的 -3dB 偏移约 3%。飞控参数本来就是手调的，3% 无所谓。
3. **惯性**：PT1 Euler 是 Betaflight 的历史实现，整个生态（调参教程、PID 推荐值）都基于它。换成 Bilinear 会让所有已有推荐参数失效。

### 为什么 Riccati 只迭代 150 次？

Riccati 方程（Kalman 的设计方程）从任意初始值迭代收敛到稳态增益。对 2×2 矩阵：

- 10 步：误差 $\sim 10^{-3}$
- 50 步：误差 $< 10^{-10}$
- 150 步：误差 $< 10^{-40}$（远超双精度浮点极限）

没有理由用更多步。计算代价 ≈ 0.1ms。

### 为什么二分搜索 40 步而不是解析解？

LKF 的 -3dB 频率与 $r$ 的关系没有简单的解析表达式（因为 Riccati 方程是非线性的）。直接解析反求 $r$ 需要求解矩阵 Riccati 方程的逆——数学上可能但代码复杂。

二分搜索 40 步 = $\log_2(10^5/10^{-5}) \approx 33$ 位精度，远超 spinbox 5 位小数分辨率。计算代价 = 40×150 次 2×2 矩阵运算 ≈ 3ms。"够好且够快"就是最好的工程方案。

## 常见误解纠正

> 参考 LoRa 文档风格：直接列出"你可能以为的 vs 实际的"。

### ❌ "Kalman 滤波器是非线性滤波器"

**实际**：Kalman 滤波器在**设计阶段**（Riccati 方程）是非线性迭代。但一旦收敛得到 K，**运行时**就是固定系数的线性差分方程，和 PT1 一样。本分析仪的 Bode 图能直接比较它们，正是因为运行时都是 LTI 系统。

### ❌ "LKF 的 DC 增益可能不是 1"

**实际**：Kalman 滤波器是**无偏估计器**，DC 增益恒等于 1.0——这是数学性质，不受参数影响。你在幅频图上看到的"增益>1"是**谐振峰值**（H=[1,1] 模式下 2 阶系统的共振），不是 DC 增益。

### ❌ "Bilinear 比 Euler 好，飞控应该用 Bilinear"

**实际**：Bilinear 精度更高，但计算量翻倍。在 32kHz 循环率、几十个滤波器级联的飞控中，Euler 的 3% 频率偏移完全可接受，且整个生态（调参推荐值）都基于 Euler。换算法 = 废掉所有已有调参经验。

### ❌ "二分搜索是一种蒙特卡洛方法"

**实际**：二分搜索是**确定性算法**，要求目标函数是单调的。每步精确砍半搜索空间，O(log n) 收敛。蒙特卡洛是随机采样，用于高维/组合空间（如博弈树）。唯一共性是"搜索"二字。

### ❌ "PSD 就是 FFT 取模方"

**实际**：单个 FFT 取模方的结果方差极大（噪声性质），且数值依赖信号长度。Welch 法把信号分段、各段 FFT 取模方再**平均**，并**归一化到每 Hz 功率**。结果平滑、可重复、不依赖信号长度。

### ❌ "H(z) 和 DEQ 是两种不同的滤波器表示"

**实际**：是**同一件事的两种写法**。H(z) 是频域分式，DEQ 是时域递推。它们用**完全相同的 b,a 系数**。`poly_z_str(b)` 写出 H(z) 的分子，`diff_eq_str(b,a)` 写出差分方程——输入一样。
