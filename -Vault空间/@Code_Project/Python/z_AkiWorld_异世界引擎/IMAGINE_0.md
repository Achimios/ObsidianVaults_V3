# VOLUME I — 数学战场

## 核心顿悟：Perlin 的真正用法

UE5 里 Perlin 生成静态地形 → 千篇一律，Erosion 和 Distribution 也救不了，必须大量手改。

**根本原因**：眼睛有时间识别 Perlin 的各向同性对称性（"那个对角线纹路又出现了"）。

**Perlin 真正擅长的**：**动态地图** — 时间维度打开一切。

```
P(x, y, z, t)   ←  加了 t 就完全不同了
```

- 等值面 `P = c` 随 t 移动 → 地形"呼吸"，岛屿涌现又消融
- 变形场 `(dx, dy, dz) = ∇P(x,y,z,t)` 推动顶点 → 多边形拓扑不变，形状持续流动
- **curl noise** 保证体积守恒（不撕裂），适合流体感折叠空间

![[Terrain Evolve.mp4]]

参考视觉：《奇异博士》折叠城市 = 多层 fBm + 高权重 curl noise。**定格反而假，动起来就是"对的"。**

### 分形的加成

fBm（fractional Brownian motion = 分层叠加的 Perlin）给出多尺度自相似：
放大看细节、缩小看整体，运动在所有频率层同步流动 → 极度有机。

### 体素 + 多边形组合架构

```
Perlin/fBm 标量场(t)
       ↓
  Marching Cubes 体素转网格   ← 拓扑真正需要变化时（洞开合、分裂）
       ↓
  多边形 Mesh
       ↑
  顶点位移 Displacement    ← 实时动画首选（不重跑 MC，快得多）
```

### 1. 热传导寻路 — 追击与躲避的物理直觉

地图是一个动态温度场。敌人是热源，你是冷点。

$$\frac{\partial u}{\partial t} = \alpha \nabla^2 u + S(\text{enemies}) - S(\text{obstacles})$$

- 热量在空间中扩散，穿不过墙，绕得过拐角
- 梯度 $-\nabla u$ 就是**最优逃亡方向**——不需要 A*，连续可微，天然避开死角
- 追击方沿 $+\nabla u$ 走 → 自动形成包围圈（等温线收缩）
- 障碍物 = 热绝缘体 → 角落里热量积聚 = "危险区域"的物理直觉

**玩家直觉训练**：100小时后你感受到的不是箭头，而是**空间的冷热**。

---

### 2. 拓扑出口 — 同调群是逃生地图

![[PIC_topology_holes_manifold_escape_route_3d_donut_torus_betti_numbers_glowing_blue.jpg]]

地形是一个动态 3-流形。洞 = 非平凡的一阶同调类 $H_1$。

```
环境拓扑分析 → Betti数 β₁ = 洞的数量
β₁ = 0  →  没有环路，无法逃脱包围圈
β₁ ≥ 1  →  存在拓扑出口，找到生成元就是逃生路
```

- **地形变形时**：Marching Cubes 实时更新同调群，洞开洞合
- **一个被包围的环面**有两个独立逃生回路（经线 + 纬线），玩家学会辨认"哪个维度的洞还开着"
- 三维空间的闭合曲面可在四维中**平移旋转**，投影回三维出现新洞 → **这就是飞船的维度跃迁技能**

### 3. 雅可比奇异点陷阱 — 让敌人撞墙

飞船运动方程：$\dot{\mathbf{q}} = J(\mathbf{q}) \cdot \mathbf{u}$，其中 $J$ 是雅可比矩阵。

$$\det(J(\mathbf{q}^*)) = 0 \implies \text{奇异点：某些方向无法运动}$$

- 引诱敌人进入地形的奇异配置区 → 敌机运动方程退化 → **被动力学锁死，无法转向**
- 分叉点（Bifurcation point）附近：极小的扰动 → 两条完全不同的轨迹
- 玩家技能：**在敌人预测路径上注入微小噪声场** → 敌人轨迹走向另一个分支，撞上动态生成的山坡
- 引擎过载：当飞船在奇异点附近强行施加推力 → 控制量趋于无穷 → 爆炸

> [!NOTE]
> Akimos补充：不如根据敌人当前位置和动量，进行敌方无法避开的地图变换，或更复杂，在弹道轨迹甚至行为树上创造 敌方运动学方程无法避开的奇异点。之所以要用雅可比矩阵而非暴力输出，就是为了实现，最小代价cost。
> 还得考虑敌方武器、运动性能和技能
> 并且是在 博弈场景 和 有限观测 情况下进行推断
> 为了让玩家有时间发挥`数学直觉`而非单纯的`搏斗直觉`，可以发动 slow-mo Focus View，配合相机动画，列出敌方属性、运动预测和决策树，我方黑客属性越强，对敌方的预测和观测越全，甚至AI能给出更优的建议

### 4. 主动造路 — 数学变换作为技能

绝对死路下，玩家激活**变换引擎**，直接修改局部 Perlin 场：

| 技能       | 数学操作                   | 效果               |
| -------- | ---------------------- | ---------------- |
| **裂谷**   | 在标量场局部加高斯脉冲 $\delta P$ | 等值面破开，出现新通道      |
| **迷雾陷阱** | 注入高频噪声到敌人传感器范围内的场      | 敌人热传导地图失真，追踪路径偏转 |
| **折叠**   | 局部仿射变换 $P \to AP + b$  | 地形片段绕轴旋转，把墙变成路   |
| **奇点炸弹** | 在指定区域插入雅可比奇异配置         | 进入的敌机运动自由度降至零    |
| **相变触发** | 将局部 Perlin 参数推过分叉阈值    | 地形在该区域发生相变，拓扑突变  |

### 5. 分形预判 — 地图的自相似性作为先验

fBm 地形是多尺度自相似的：放大看细节和缩小看整体，结构是同族的。

- **训练玩家直觉**：见过小尺度的峡谷，就能预判大尺度的路网走向
- **分形维度**作为战术参数：$D \approx 1.7$ 的地形 → 逃亡空间分散；$D \approx 2.3$ → 密集迷宫，有利于伏击
- 随时间演变的 $P(x,y,z,t)$：短时预测 = 惯性外推；长时预测 = 分形先验 + 混沌误差增长

### 6. 洛伦兹吸引子 — 玩家心理即游戏物理

最深的设计不在关卡，在**玩家大脑本身**。

玩家的注意力 / 决策状态是一个动力系统：

$$
\dot{x} = \sigma(y-x), \quad \dot{y} = x(\rho - z) - y, \quad \dot{z} = xy - \beta z
$$

- 高压战斗 = $\rho$ 增大，系统离开稳定点进入混沌盘旋 → **心流状态**
- 太简单 = $\rho$ 过低，轨迹死在不动点 → **无聊**
- 信息过载（随机白噪音注入）= 吸引子结构崩塌 → **崩溃**
- **游戏的目标**：把玩家的心理参数持续维持在洛伦兹混沌区域的边缘——感性引导的最高神经网络秩序

> 玩家不是在学数学。玩家的神经网络**正在成为**这些数学对象的肉身实现。

## 场方程层 — 空间本身是物理的

### 黎曼度量张量 $g_{\mu\nu}$ — 游戏的宪法

所有物理都从这里导出。地图**不是**一个欧氏空间，它是一个带有度量的流形：

$$ds^2 = g_{\mu\nu}(x, t) \, dx^\mu \, dx^\nu$$

- **距离** = 度量张量积分，两点之间"近"或"远"取决于局部曲率
- **直线** = 测地线方程 $\ddot{x}^\mu + \Gamma^\mu_{\nu\sigma} \dot{x}^\nu \dot{x}^\sigma = 0$（克里斯托弗符号）
- **改变度量 = 改变整个物理法则**：飞船引擎技能可以局部扭曲 $g_{\mu\nu}$ → 让"直线"变成绕过墙的曲线

玩家在不知不觉中学会：**直觉上的"近路"取决于空间本身的形状，而不是欧氏距离。**

在弯曲时空中的协变形式：$\nabla_\nu F^{\mu\nu} = J^\mu$

$$i\hbar \frac{\partial \psi}{\partial t} = \hat{H} \psi = \left(-\frac{\hbar^2}{2m}\nabla^2 + V(\mathbf{x},t)\right)\psi$$

- **观测 = 波函数坍缩**：敌人的雷达扫描 = 对你施加位置算符测量，坍缩你的概率云 → 暴露位置
- **隐身技能 = 维持叠加态**：保持你的波函数扩散在大范围内，$|\psi|^2$ 低但不为零 → 敌人无法确定位置
- **量子隧穿**：波函数有非零振幅穿过势垒（墙） → 低概率穿墙，作为极限逃脱机制
- **纠缠武器**：发射纠缠粒子对，测量一个 → 瞬间确定另一个的位置（对敌人施加）

**玩家直觉训练**：你在学**量子力学的测量理论**，用的方式是躲避雷达扫描。

### 纽结敌人 — Reidemeister 解构术

**场景**：空间中漂浮的能量绳结，就是数学意义上的**纽结**。三叶结、八字结、环面纽结……每种拓扑类型都是不同的敌人种类。

纽结不可能被"切断"——但可以被**变形等价归零**（解纽结 = 化为平凡结 = 消灭敌人）。

三种 Reidemeister 移动对应三种技能：

| 移动 | 操作 | 游戏动作 |
|---|---|---|
| **Ⅰ（扭转）** | 增加/消除一个自交叉 | 旋转光束扫过结的局部 |
| **Ⅱ（滑过）** | 一股绳滑过另一股下方 | 相位平移弹（时间延迟脉冲） |
| **Ⅲ（三角）** | 绳在三角区域内平移 | 空间平移脉冲，推动一段绳 |

**纽结不变量**就是"血量"的本质：Alexander 多项式越复杂，需要越多步移动。三叶结 $3_1$ vs 环面纽结 $T(p,q)$ — 后者是精英怪，因为它需要更深的代数分解。

**UI机制**：3D投影视图 + 高亮交叉点，点击选择 Reidemeister 移动类型，鼠标拖拽施加力。HUD 显示当前纽结的 Alexander 多项式（血量条）。

> 直觉：纠缠是可以解的——但要知道用哪种移动。

### KAM 不变环面 — 混沌海的避风岛

**物理基础**：在近可积哈密顿系统中，大多数不变环面在小扰动下**存活**（Kolmogorov-Arnold-Moser 定理）。只有共振环面会破碎，变成岛链和随机层。

$$\text{KAM 条件}: \left|\mathbf{k} \cdot \boldsymbol{\omega}\right| > \frac{C}{|\mathbf{k}|^\tau} \quad \forall \mathbf{k} \in \mathbb{Z}^n \setminus \{0\}$$

数**"足够无理数"**的频率比 $\omega_1/\omega_2$ 对应的环面最稳定——黄金比例 $\phi = (1+\sqrt{5})/2$ 是最难被共振摧毁的。

**游戏机制**：
- 相空间地图上显示庞加莱截面，稳定环面 = 同心椭圆（安全），随机层 = 散点（危险）
- 玩家飞行在相空间中：只要轨迹是 KAM 环面，就不会迷失；一旦偏入随机层，轨迹变混沌
- **敌人**住在共振岛链（Poincaré-Birkhoff 岛链），那里是确定性混沌的交界，最难清理
- 更高扰动强度 → 更多 KAM 环面被摧毁 → 世界变得越来越混沌 → 难度压力

**无理数武器**：使飞行频率比精确等于 $\phi$ 的时间内，护盾最强（黄金比例时刻）。

**UI机制**：庞加莱截面小HUD，显示当前轨迹点的稳定性色彩（蓝=安全，红=混沌）。

### 孤波渡关 — 与 KdV 孤子共舞

**物理基础**：KdV 方程（浅水波，等离子体，光纤脉冲）：

$$\partial_t u + 6u\,\partial_x u + \partial_{xxx} u = 0$$

孤波解：高振幅孤子跑得快，低振幅跑得慢。两个孤子相遇，**彼此穿过**，只有相位漂移（不损耗、不融合、不反射）——这是完全可积系统的奇迹。

**游戏机制**：
- 关卡中漂浮着多列孤子波墙，每列振幅不同（速度不同）
- 玩家飞船不能穿越孤子（碰撞 = 受力）
- **策略**：利用孤子相互穿越的特性，在两波孤子即将重叠的瞬间，借助重叠区的"相位漂移"位移飞出
- 配合 **Chrono Charge（V.A.T.S. 慢镜头）** 预判孤子交叉时间窗口
- **进阶**：破缺 KdV 可积性（加小扰动项）→ 孤子不再完美穿越，开始辐射能量，关卡复杂化

**UI机制**：侧视图波形 HUD，显示孤子位置、速度、振幅实时预测。

> 完全可积 = 无摩擦的宇宙。扰动它，就有了死亡。

### 椭圆曲线引力收割 — 群律即物理

**数学基础**：椭圆曲线上有群律：过曲线上 $P$、$Q$ 两点作直线，第三个交点关于 $x$ 轴的镜像即为 $P+Q$。

$$E: y^2 = x^3 + ax + b$$

这不只是代数——这是引力场的几何。

**游戏机制**：
- 空间中有两个"引力共振点" $P$ 和 $Q$（实际上就是椭圆曲线上的两点）
- 资源（能量晶石）只在满足 $P+Q+R = O$（曲线上共线三点，和为单位元）的位置 $R$ 处汇聚
- 玩家需要**反推**：我要在哪里等，才能让 $P + Q$ 的群律落在我这里
- 敌方保护着"零元" $O$（无穷远点）——摧毁 $O$ 点防御，整条曲线的群结构紊乱，所有资源点同时涌现

**进阶**：超奇异椭圆曲线——配对映射（Weil pairing）在敌我双方的作战系统中充当"密钥交换"，伪造配对 = 劫持敌方通信网络。

### Berry 相位环路充能 — 路径刻进几何

![[Pasted image 20260426180422.png]]

**物理基础**：量子态沿参数空间闭路绝热演化一圈，回到原位，却积累了一个纯几何相位：

$$\gamma_n = i\oint \langle n(\mathbf{R})|\nabla_{\mathbf{R}}|n(\mathbf{R})\rangle \cdot d\mathbf{R}$$

这个 Berry 相位**只取决于路径在参数空间包围的面积**（类 Stokes 定理），与演化速度无关。这是几何的礼物，而非动力学的代价。

**游戏机制**：
- 玩家在**参数空间**（非物理空间）飞行环路，累积 Berry 相位电荷
- 不同闭路形状 → 不同相位值 → 解锁不同技能（$\gamma = \pi/2$ 解锁四分之一翻转，$\gamma = \pi$ 解锁镜像攻击，$\gamma = 2\pi$ = 回到原态，什么都没发生）
- **费曼费恩曼直觉**：绕一圈零能耗，却改变了内在态——这是**纯拓扑的能量储存**
- 敌人：磁单极子（参数空间的 Dirac 弦），绕过单极子的路径会产生额外不连续相位，需要绕行

**UI机制**：参数空间 2D HUD，显示当前路径和包围面积，Berry 相位电量条。

> 不是你走了多远，是你包围了什么。几何是真正的货币。

### 谱图刺杀 — 切断 Fiedler 向量

**数学基础**：图 Laplacian 矩阵 $L = D - A$（度矩阵减邻接矩阵）的第二小特征值 $\lambda_2$（代数连通度）衡量图的"切断难度"。

对应特征向量 $\mathbf{v}_2$（**Fiedler 向量**）给出最优二分割方向：$v_{2,i} > 0$ 的节点放一边，$v_{2,i} < 0$ 的放另一边。

**游戏机制**：
- 敌方蜂群 = 一张图，每个成员是节点，通信链路是边
- HUD 自动计算实时 Fiedler 向量，高亮**最小割边集**
- 只需消灭 Fiedler 向量分量绝对值最大的几个"关键节点"，网络代数连通度 $\lambda_2 \to 0$，蜂群分裂成两个不互通的子群
- 更高效的打法：不打最多连接的节点（度中心性），打 Fiedler 中心性最高的节点
- **瓶颈边攻击**：找到边 $(i,j)$ 使得删除后 $\lambda_2$ 下降最大（谱二分的"切割价值"最高的单边）

**进阶敌人**：Ramanujan 图（$\lambda_2$ 非常接近 $2\sqrt{d-1}$，接近理论最优 expander graph，极难分割——"几乎不可摧毁的通信网络"）。

**UI机制**：图结构 HUD 叠加在蜂群实体上，Fiedler 向量热图着色，FPS 狙击风格打关键节点。

### 尖点灾变 — 地形的记忆与突跳

**数学基础**：René Thom 突变理论。尖点突变（cusp catastrophe）的势函数：

$$V(x; a, b) = x^4 + ax^2 + bx$$

平衡点 $\partial V/\partial x = 0$ 构成折叠曲面，在"燕尾"内部是三个局部极值（两稳一不稳）。当参数 $(a, b)$ 沿路径穿越折叠线时，系统**突跳**到另一个稳态——且有**滞后**：正方向穿越和反方向穿越的突跳点不同。

**游戏机制**：
- 地形存在"尖点区域"，该区域有两种稳定形态（高地形 / 低地形）
- 飞行速度（参数 $b$）和高度（参数 $a$）决定地形处于哪个分支
- **突跳陷阱**：快速飞过尖点区域 → 地形突然从高平台崩塌为深渊（参数穿越折叠线）
- **滞后武器**：利用滞后性，把敌人引入已突跳的分支，再自己从另一路径安全绕过（敌我的参数路径不同，突跳时机不同）
- **折叠线**可以在 HUD 上可视化，提前预测突跳位置

**其他突变类型**：折叠突变（fold，1D，突然消失）、蝴蝶突变（butterfly，高阶，更多稳态）构成不同关卡地貌。

### p 进隐身 — 距离度量的颠覆

**数学基础**：p 进赋值：$|p^n \cdot m|_p = p^{-n}$（$\gcd(m,p)=1$）。

这意味着：在 p 进度量中，**数越能被 $p$ 整除，就越接近零**。$p^{100}$ 在实数里是天文数字，在 3 进度量里几乎就是 0。

超距离不等式：$|x + y|_p \leq \max(|x|_p, |y|_p)$（比三角不等式更强）——这让 p 进空间的"球"有奇异嵌套结构。

**游戏机制**：
- 某些关卡空间同时存在**实数度量**（物理坐标）和 **p 进度量**（信号坐标）
- 敌方导弹用 p 进度量追踪——它们的"最短路径"不是实空间的直线，而是 p 进测地线（看起来绕了大弯的奇怪路径）
- **隐身技**：让自己的坐标变成高度可被 $p$ 整除的数（比如 $x = p^{50} \cdot q$），在 p 进度量下距离敌人"极近"但在实空间中可能相距甚远——导弹的追踪优化会误判
- **武器**：在敌人 p 进度量最小的方向（实数意义上远，p 进意义上近）投弹

> 同一个宇宙，两种距离。选择哪个是真实的，那就是你的武器。

> Akimo 想法：穿越敌方防御网络，或在 攻击波束阵列、飞弹阵列中动态穿梭

### 持续同调扫描仪 — 拓扑的信号与噪声

![[PIC_persistence_diagram_barcode_birth_death_pairs_signal_noise_dark_visualization.jpg]]

**数学基础**：Persistent Homology — 在一系列阈值 $\varepsilon_0 < \varepsilon_1 < \cdots$ 下，计算点云/场的贝蒂数（$\beta_0$=连通分量，$\beta_1$=一维孔洞，$\beta_2$=空腔）。

每个拓扑特征有 birth-death 对 $(b, d)$，**长寿命特征**是结构性的，**短寿命特征**是噪声。

**游戏机制**：
- 玩家飞船配备**持续同调扫描仪**，实时分析周围 Perlin 地形的拓扑特征
- 显示 persistence diagram：长寿命的点（高 $d - b$）= 真实通道/洞穴（可以穿越），短寿命的点 = 地形噪声（假通道，硬碰会被挤碎）
- **敌方伪装**：部分敌人是"拓扑噪声"形态（极短寿命拓扑特征），肉眼看似真实通道，扫描仪一扫即露
- **瓶颈定理**：两个点云之间的持续同调距离（interleaving distance）决定多少能量可以穿越两个战区间的"拓扑隧道"

**UI机制**：弧形扫描仪 HUD，persistence diagram 叠加图（$x$轴=birth，$y$轴=death，对角线上方的点颜色越亮=寿命越长=越真实）。

## 绝热演化走廊 — 耐心是最强的数学

**量子绝热定理**：系统在慢变哈密顿量下，始终留在瞬时本征态（基态）：

$$\text{绝热条件}: \frac{|\langle 1|\dot{H}|0\rangle|}{(E_1 - E_0)^2} \ll 1$$

间隙 $\Delta = E_1 - E_0$ 越大，可以走得越快；间隙越窄，必须更慢。

**游戏机制**：
- 狭窄的能隙走廊：地形两侧是激发态势垒，玩家必须足够慢才能不跃迁
- 间隙测量仪 HUD 显示当前 $\Delta$，速度上限实时变化
- **反绝热捷径（COLD）技术**：主动施加反绝热哈密顿量 $H_\text{CD} = i\hbar(\partial_t |n\rangle\langle n| - \langle n|\partial_t |n\rangle)$，可以快速通过同时补偿跃迁——消耗大量 Chrono Charge，但允许高速穿越
- **与 KAM 联动**：绝热慢行时更容易维持 KAM 环面（小扰动 = 更多不变量）

> 时间是代价。而慢，是最难的技能。

## 重整化群 — 战略缩放即武器

**物理基础**：RG flow：沿 RG 流方向，**无关算子**指数衰减，**边缘算子**对数变化，**相关算子**指数增长。不同尺度看到的"有效理论"不同。

$$\beta(g) = \mu \frac{dg}{d\mu} \quad \text{（耦合常数随能标的流动）}$$

固定点（$\beta = 0$）= 相变临界点 = 最危险的地方（标度不变，无特征长度）。

**游戏机制**：
- 玩家可以主动切换**观测尺度**（UV/IR）
  - **UV（高分辨）**：看到所有短波细节，但战略格局消失，被细节淹没
  - **IR（低分辨）**：看到全局流向，无关算子消失，只剩最相关的威胁
- **相关算子**在 IR 视角下变成最明显的危险（放大），在 UV 下反而混在噪声里
- 固定点附近：尺度不变性 → 敌方所有特征频率都出现 → 最难应对，但同时玩家自身也变成"标度不变武器"
- 共形对称敌人：只在固定点附近存活，离固定点越远它们越弱

**UI机制**：能标滑杆（UV ↔ IR），场景细节随尺度实时LOD变化，相关算子用颜色标注。

## 可玩性三元论
![[Pasted image 20260426200129.png]]

### 可玩性（Playability）

**本质**：how many maneuvers you can do, and provide some options that you never imagined you could do.

不是技能数量，不是地图大小，是**决策分叉的密度与意外性**。玩家的选项空间越密、越出人意料，可玩性越高。

### 成就感（Achievement）

**本质**：wonderful or hard maneuver → payment → harvest，且有美感的收获反馈系统。

成就感系统的完整链路：
```
操作难度/美感 × 成功率
        ↓
    收获（数值/叙事/听觉/视觉）
        ↓
    认可追捧（UI 动效 + 音效 + 文字 + 画面 + 其他玩家）
```

关键：**是带美感的，曲线合理的，阶梯收获的，由自我兴趣驱动的。不是学校数学考试的方式。**

### 耐玩性（Replayability）

**本质**：决策树路径演化的可能性 × 不同演化之间的非同质度。

参照系：围棋（规则极简，分叉无穷）、Dota（角色差异，路径非同质）、新维加斯（叙事分叉 + 世界响应）、战地（高频混沌）、我的世界（玩家写规则）。

**超预期收获**是耐玩性的顶峰：不是一个你不知道能做到的操作，而是**一个你早就知道能做、但永远不知道做成功那一刻是什么感觉**的操作。

> 世界的丰富度取决于可交互度（自我深度 × 世界丰度）。  
> 世界的大小取决于路径代价，而非空间大小。  
> 一个 10m × 10m 的废弃工厂，可以比一个 10km 的开阔地图更大。

## 起点：废墟工厂

> 最大约束产生最纯粹的玩法基因。

![[Pasted image 20260426204423.png]]

起步无需异世界。一个**废弃工厂**：

```
几何约束：非规整的墙壁、钢柱、走廊断面
路径约束：碎玻璃、倒塌的楼板、生锈的门
视觉语言：废墟感 = 路径约束 + 路径自由 同时存在
噪声：几何噪声（不规则结构）+ 视觉噪声（锈蚀 + 蒸汽 + 光线衰减）
```

地图大不是因为米数大，而是因为**每一米都有决策分叉**：
- 这块盖板能不能踩（物理）
- 这扇门后有什么（信息）
- 那根管子能不能攀（运动学）
- 这里够不够暗来一次暗杀（光照 × 敌方视野）

## 世界生成法则

### 敌人分布

```
视野覆盖优化：敌人位置分布 ≈ 视野函数对空间的覆盖度最大化问题
重要地点权重：关键节点（保险柜、出口、武器库）周围密度更高
最优主轴 + 意外偏离：
  - 重甲兵"应该"在保险库门口（最优解）
  - 但今天可能在楼下和轻步兵抽烟散步（偏离主轴的噪声）
  - 这个偏离就是耐玩性的关键
```

### NPC 世界噪声

```
个体决策风格（连续分布，非离散种类）：
  疯狂度 ↔ 冷静度
  勇敢度 ↔ 谨慎度
  慵懒度 ↔ 警觉度
  → 这些参数的组合决定行为，且有解析解不存在的区域（决策风格参数空间的混沌区）

个性（低概率但高影响）：
  某个 NPC 喜欢音乐 → 遇到特定音效会停止行动 0.8s
  某个 NPC 妻子生病住院 → 精神状态不稳定，反应时间方差更大
  某个 NPC 之前当过厨子 → 厨房区域他更有安全感，warning threshold 更高
```

**高参世界本身无解析解**。这些偏离理性人的 NPC 不是 bug，是涌现的基础。

## 三种战斗节奏

> 玩家在开始关卡前选择音乐模式，决定信息密度与节奏。这不是难度选项，是**玩法风格选择**。

### ♩ 汉斯·季默模式 — 全局视野战场

- 全局视野开启：实时已知所有敌方位置（像 X-ray 一样）
- 高频主动世界：敌人在移动、交流、切换站位
- **暴力暗杀最顺畅**：预先看到每个人，设计连锁击杀路线
- **失败即切换**：暗杀失败 → 无缝过渡到**高混沌战场模式**（Nirvana — 噪声、混乱、肾上腺素）
- 成就感来自：**计划的完美执行** + **混沌中的即兴应对**

### ♩ 莫扎特模式 — 信息博弈

- 战争迷雾开启，敌方位置不可知
- 可以**付出代价**（时间、资源、风险）换取**部分**敌方位置信息
  - 电子扫描：消耗资源，显示 5s 内特定区域敌方轮廓
  - 声音三角测量：根据声音来源估算位置，有误差分布
- 成就感来自：**信息的精确运用** + **用不完整信息做出最优决策**
![[Trace Blink.mp4]]

### ♩ 舒伯特模式 — 战争迷雾潜入

- 完全无扫描，对敌人位置完全不可知
- **迷雾是游戏乐趣的本体**，不是障碍
- 路径、光照、声音管理是核心机制
- 成就感来自：**在不确定性中精确行动** + **没有任何人发现你存在过**

### 阶 0 — 原质（最大约束）


## 成就感工程 — 收获曲线设计

> 成就感不能是平的，也不能是垂直墙。它是一条有内容的斜率。

```
普通操作：            [子弹击中] 微小音效，轻微视觉反馈
良好操作：            [爆头] 击杀音效，轻微慢镜头
优秀操作：            [连锁暗杀] 电影镜头，专属音乐节
漂亮操作：            [穿越量子概率云后弹道完美命中] 全屏慢镜头 + 方程叠加显示 + AI同伴惊叹
传说操作：            [在范畴消融前最后一击] 世界开始透明，BGM 变 Bach，白屏
```

**AI 同伴对成就感的放大**：不是系统提示"优秀操作"，而是她真正说出一句话，那句话只有在这个具体操作发生后她才会说。**语境性认可**远比徽章更有力。
