## 5D 演示（`维度_5_堆积球.py`）专属说明

### 三个滑条

| 滑条 | 作用 | 范围 |
|------|------|------|
| z | 固定 z 坐标 → 左图切出 xy 截面 | [-1.5, 1.5] |
| w | 固定 w 坐标 → 5D→4D，影响哪些角球出现 | [-1.5, 1.5] |
| v | 固定 v 坐标 → 5D→4D，与 w 共同决定右图 3D 截面 | [-1.5, 1.5] |

右图展示的是在 (w, v) 固定时，xyz 三维空间里各球的截面球体。

### 5D 核心数学亮点

**中央球半径 r = √5 − 1 ≈ 1.236 > 1**（超出立方体半宽！）

| 状态 | 现象 | 原因 |
|------|------|------|
| w=0, v=0 | 中央球"突破边界"，角球全消失 | 角球截面半径 = 0 |
| w=1, v=1 | 8 个 xyz 角球满尺，中央球缩小 | 中央球截面半径 = √(r²-w²-v²) 变小 |

## 球堆积核心数学

### 为什么中央球在高维变大？

```
3D 版本：
  - 2×2×2 立方体（角在 ±1）
  - 8 个角球，r=1，球心在 (±1, ±1, ±1)
  - 中央球 r = √3−1 ≈ 0.732
  验证：角球心到原点 = √3 = 1 + (√3−1) ✓ 正好相切

4D 版本：
  - 16 个角超球，r=1，球心在 (±1, ±1, ±1, ±1)
  - 中央超球 r = √4−1 = 1（与角超球等大！）
  验证：角超球心到原点 = √4 = 2 = 1 + 1 ✓

5D 版本：
  - 32 个角超球，r=1
  - 中央超球 r = √5−1 ≈ 1.236 > 1（超出立方体！）
```

**规律**：N 维中央超球半径 = √N − 1。N≥10 时中央超球已比角球大得多，且 N>9 时突破立方体边界。

### 4D 切片数学（滑条的意义）

对于一个 4D 超球（球心 c₄，半径 r）与超平面 **n₄·x = t** 的交截：

$$d = \mathbf{n}_4 \cdot \mathbf{c}_4 - t \qquad r' = \sqrt{r^2 - d^2}$$

截面是 3D 球，半径 r'，球心在超平面上 c₄ 的投影处。改变 **切面方向 n₄** = 改变切开 4D 形状的角度（不是相机旋转！）。
