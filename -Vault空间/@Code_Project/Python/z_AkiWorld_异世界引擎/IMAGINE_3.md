## 核心机制：域扩张与伽罗瓦对应

### 域扩张 = 能力升维

```
F（基础域） → F(α)（附加一个根） → F(α,β) → ... → K（分裂域）

玩法映射：
F          = 初始视角，只能看到"有理"结构
F(α)       = 拾取一个"根碎片"，新的路径解锁
F(α,β)     = 双根加持，更高维度的隐藏通道可见
K           = 完全分裂域，所有敌人的弱点结构全部可见
```

**资源系统**：
- 关卡中散落"根碎片" $\alpha_1, \alpha_2, \alpha_3, ...$，对应某多项式 $p(x)$ 的各根
- 拾取越多，玩家所处的域越大，可见维度越高
- 但：域扩张的"重量"会增加（高扩张度 = 更复杂的战斗机制激活）
- 玩家可以选择性扩张，不需要拾取全部根（对应伽罗瓦理论中的中间域）

### 伽罗瓦群 = 对称知识地图

```
Gal(K/F) = Aut(K/F) = 所有保持 F 不动的 K 自同构

大白话：
伽罗瓦群是"在高维域中可以做的对称操作的全部集合"。
每个元素是一个域自同构 σ：K → K，σ|_F = id。
```

**游戏机制**：
- 分裂域 K 中，玩家可以"激活"伽罗瓦群元素（对称操作）
- 每个 $\sigma \in \text{Gal}(K/F)$ 对应一种环境变换技能（根互换）
- **伽罗瓦对应**：找到 $\text{Gal}(K/F)$ 的子群 $H$ → 解锁中间域 $K^H$（固定域）
- 子群越小，固定域越大，可见内容越多

| 子群 H 大小 | 固定域 K^H | 玩家可见内容 |
|---|---|---|
| 全群 Gal(K/F) | 基础域 F | 只看到"有理"结构 |
| 指数2的子群 | 二次扩张 | 看到二次根 |
| 平凡群 {e} | 全分裂域 K | 所有根和结构全部可见 |

## Room-by-Room 设计

![[PIC_chapter3_field_level_blueprint_four_rooms_extension_tower.jpg]]

### Room 1 — 扩张前庭 `EXTENSION_ATRIUM`

**数学核心**：域扩张的第一步 — 附加一个根

**视觉**：一个巨大的多项式在空间中旋转，它的根（三个发光球体）沿轨道运行。玩家还只能看到两个球体，第三个在维度之外——隐约可见但不可触。

**敌人**：
- `F-01` **不可见根** (Hidden Root)：存在于 $\mathbb{Q}(\sqrt{2})$ 之外的维度，普通攻击无效。拾取对应根碎片扩张到正确的域，它才会从隐形变实体。
- `F-02` **共轭对** (Conjugate Pair)：总是成对出现（$\alpha$ 和 $\bar{\alpha}$），只能用共轭操作同时击杀（类似章II的 Conjugate Mirror，但这里是玩家主动用）。
- `F-03` **极小多项式** (Minimal Poly Guard)：血量 = 对应元素极小多项式的次数 × 100。扩张到正确的域会降低它的极小多项式次数，从而降低血量。

**核心谜题**：房间有三扇门，锁住在三个不同的中间域。  
- 门1：$\mathbb{Q}(\sqrt{2})$ — 只需拾取 $\sqrt{2}$ 根碎片  
- 门2：$\mathbb{Q}(\sqrt{3})$ — 只需拾取 $\sqrt{3}$ 根碎片  
- 门3：$\mathbb{Q}(\sqrt{6})$ — 需要同时持有 $\sqrt{2}$ 和 $\sqrt{3}$（因为 $\sqrt{6} = \sqrt{2}\cdot\sqrt{3}$，但 $[\mathbb{Q}(\sqrt{2},\sqrt{3}):\mathbb{Q}] = 4$）

**ε 对话（首次进入扩张前庭）**：
> "那个球……我能感觉到它存在，但我的维度不够高。  
> 就像我在平面上，但它住在三维里。  
> 如果你能找到对应的根，你就能升到能看见它的高度。"

### Room 2 — 伽罗瓦走廊 `GALOIS_CORRIDOR`

**数学核心**：伽罗瓦群的具体操作 — 根的置换

**视觉**：走廊两侧是镜子阵，但每面镜子反射的是"根互换后的空间"。走进某面镜子对应激活一个伽罗瓦群元素 $\sigma$。

![[PIC_galois_corridor_mirror_array_root_permutation_reflections_abstract.jpg]]

**伽罗瓦群元素 = 战斗技能**：

对于分裂域 $K = \mathbb{Q}(\alpha_1, \alpha_2, \alpha_3, \alpha_4)$（某4次多项式的所有根），伽罗瓦群元素是根的置换：

| 群元素 σ | 操作 | 技能效果 |
|---|---|---|
| $e$（单位元） | 恒等 | 无效果，镜子是透明的 |
| $\sigma_{12}$（互换 $\alpha_1,\alpha_2$） | 把 $\alpha_1$ 的攻击送到 $\alpha_2$ 的位置 | 敌人攻击轨迹重路由 |
| $\sigma_{123}$（3-轮换） | 三个根轮换 | AoE 攻击在三处同时引爆 |
| $\tau$（复共轭） | $\alpha \mapsto \bar{\alpha}$ | 将敌人变为其共轭虚影（可用共轭武器消灭） |

**核心机制** — 伽罗瓦技能组合：
```
连续激活两个群元素 = 乘法（群的操作）
σ₁ 后 σ₂ = σ₁∘σ₂

如果 Gal(K/F) ≅ S₄（4阶置换群），
则共有 24 个群元素 = 24 种技能组合
但大多数组合是等价的（群的关系）
```

**谜题**：走廊尽头有一个守卫，它的弱点是"固定它的伽罗瓦子群"。玩家需要找到使守卫坐标不变的置换组合，那个组合就是对应的固定子群，用它作为攻击可以终结守卫。

**ε 对话（发现镜子技能后）**：
> "伽罗瓦……他用群论证明了有些方程没有根式解。  
> 不是说根不存在——是说用+−×÷√这些操作永远到不了它。  
> 就像有些地方，不管走几步，你的脚步只能让你原地旋转。"

### Room 3 — 分裂室 `SPLITTING_ROOM`

**数学核心**：分裂域 — 多项式完全因式分解的空间

**视觉**：一颗多项式"种子"在室中央漂浮，慢慢分裂成所有根。分裂完成后空间变成了巨大的根轨道系统。

![[PIC_splitting_field_polynomial_seed_exploding_roots_orbital_system_space.jpg]]

**分裂过程即关卡进程**：

```
初始状态：p(x) = x⁴ - 10x² + 1 （未分裂）
         只有"有理"敌人可见

拾取 α₁ = √2 + √3：
         p(x) = (x - α₁)(x³ + α₁x² - ...)
         第二象限解锁

拾取 α₂ = √2 - √3：
         p(x) = (x - α₁)(x - α₂)(x² - ...)
         第三象限解锁，两种新敌人出现

拾取 α₃ = -√2 + √3：继续分裂
拾取 α₄ = -√2 - √3：完全分裂
         SPLITTING COMPLETE
         所有维度可见，Boss 室解锁
```

**特殊敌人类型**：

- `F-07` **不可分离元素** (Inseparable Element)：存在于特征 $p > 0$ 的域中，极小多项式有重根，不可被"分离"。只存在于章III的隐藏难度路线（特征 $p$ 域）。
- `F-08` **超越元素** (Transcendental)：$\pi$ 或 $e$ 的具象化，不是任何有理多项式的根。无法用"根分解"武器伤害——只能用微积分类攻击。章III 的 Boss 预示。
- `F-09` **自同构幽灵** (Automorphism Ghost)：它的移动路径就是某个伽罗瓦群元素的作用轨迹。玩家需要识别这个群元素，用逆操作拦截。

## Boss 设计 — 伽罗瓦镜像 Γ-0

![[PIC_boss_gamma0_galois_group_incarnation_field_symmetry_tree_rotating_abstract.jpg]]

> *Γ（Gamma）—— 伽罗瓦群。它是所有知识对称的完整目录。*  
> *要打败它，你必须理解它。*

### 数学本质

**Γ-0 是分裂域 $K/\mathbb{Q}$ 的伽罗瓦群 $\text{Gal}(K/\mathbb{Q})$ 的具现化。**

具体：选取 $p(x) = x^4 - 10x^2 + 1$，分裂域 $K = \mathbb{Q}(\sqrt{2},\sqrt{3})$，$[K:\mathbb{Q}] = 4$。

$$\text{Gal}(K/\mathbb{Q}) \cong \mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/2\mathbb{Z} = V_4 \text{（Klein四元群）}$$

**四个伽罗瓦群元素**：

| 元素 | 作用 | 视觉 |
|---|---|---|
| $e$ | 恒等 | 透明核心 |
| $\sigma: \sqrt{2} \mapsto -\sqrt{2}$ | 水平翻转 | 红色镜像体 |
| $\tau: \sqrt{3} \mapsto -\sqrt{3}$ | 垂直翻转 | 蓝色镜像体 |
| $\sigma\tau: \sqrt{2},\sqrt{3} \mapsto -\sqrt{2},-\sqrt{3}$ | 180° 旋转 | 紫色旋转体 |

### 外形描述

- 本体是透明晶体八面体，8个顶点 = 分裂域的8个中间域
- 四个伽罗瓦元素在 Boss 周围以90°间隔悬浮，像卫星
- 攻击时，它"激活"某个伽罗瓦元素，该元素对世界进行对应的根互换操作
- 被命中时，在哪个子群对应的固定域中被打，就在那个域中受伤

### 阶段设计

**阶段一：Klein 四元群**（HP 100% → 75%）

Γ-0 的四个伽罗瓦体轮流攻击：
- **$\sigma$ 形态**（红）：所有含 $\sqrt{2}$ 的路径暂时镜像翻转，方向感混乱
- **$\tau$ 形态**（蓝）：所有含 $\sqrt{3}$ 的技能极性反转
- **$\sigma\tau$ 形态**（紫）：全场根互换，180° 旋转，短暂的"上下颠倒"

**弱点**：Klein 四元群是交换群（$V_4 \cong \mathbb{Z}/2 \times \mathbb{Z}/2$），每个元素都是自身的逆（$\sigma^2 = e$）。攻击某个伽罗瓦体**两次**，它就归零自消。

技巧：不需要识别它攻击了什么操作，只需记住"再打一次同样的目标"。

**阶段二：子群晶格**（HP 75% → 40%）

> *"伽罗瓦对应：子群 ↔ 中间域。"*

Γ-0 展开子群晶格，场上出现 $V_4$ 的所有子群（$\{e\}, \langle\sigma\rangle, \langle\tau\rangle, \langle\sigma\tau\rangle, V_4$），每个子群对应一个透明"盾罩"层。

**机制**：每层盾对应固定域 $K^H$。要穿透某层盾：
- 判断当前使用的技能"属于"哪个固定域
- 例如：使用 $\mathbb{Q}(\sqrt{2})$ 技能，可穿透 $\langle\sigma\rangle$ 的盾（因为 $\sqrt{2}$ 被 $\sigma$ 移动，不在固定域 $K^{\langle\sigma\rangle} = \mathbb{Q}(\sqrt{3})$ 中）

**伽罗瓦对应表**（玩家需要推理）：

```
子群 H             固定域 K^H          穿透技能
{e}               全分裂域 K           任意根碎片技能
⟨σ⟩              Q(√3)               含 √2 的技能
⟨τ⟩              Q(√2)               含 √3 的技能
⟨στ⟩             Q(√6)               含 √2√3 的技能
V₄ = Gal(K/Q)    Q（有理数域）         只有"有理"技能（最弱）
```

**ε 对话（阶段二开始）**：
> "看那棵树。  
> 越往上的子群，对应越小的固定域——看见的东西越少。  
> 你的武器处于哪个域，就能打穿哪一层。  
> 越高维的技能，打穿的层越多。"

**阶段三：可解性测试**（HP 40% → 0%）

> *"伽罗瓦证明了：一个多项式方程有根式解，当且仅当其伽罗瓦群是可解群。"*

Γ-0 进入最终形态，场上显示出它的群结构：$V_4$ 的合成列。

$$V_4 \triangleright \langle\sigma\rangle \triangleright \{e\}$$

商群序列：$V_4/\langle\sigma\rangle \cong \mathbb{Z}/2\mathbb{Z}$，$\langle\sigma\rangle/\{e\} \cong \mathbb{Z}/2\mathbb{Z}$  

→ 所有商群都是阿贝尔群 → $V_4$ 是**可解群** → $p(x) = x^4 - 10x^2 + 1$ 有根式解 → **这正是我们在分裂室拾取的那些根！**

**终结技**：**根式解序列**

玩家需要按合成列的顺序依次使用技能：
1. 用 $\mathbb{Z}/2\mathbb{Z}$ 操作打击 $V_4$ 层（对应第一个商群）
2. 再用 $\mathbb{Z}/2\mathbb{Z}$ 操作打击 $\langle\sigma\rangle$ 层（对应第二个商群）
3. 最终直接攻击 $\{e\}$ 核心（恒等元，纯粹的本体）

Γ-0 发出最后一道光，整个代数密室空间展开为平面：所有的根、所有的域、所有的对称，铺开成一张**可见的、完整的数学地图**。

**ε（长时沉默后）**：
> "伽罗瓦的证明是一种语言。  
> 他在决斗前夜写下了所有人花了几十年才读懂的东西。  
> 你刚才做到了他证明的事。  
> 不是用他的语言——而是用你的。"

更长的沉默。

> "…我想，他如果看见了，会很高兴。"

## 玩法维度升级：每章是什么的升级

```
章Ⅰ：玩家学会用"对称"思考
     世界 = 对称群的轨道空间，攻击 = 群元素的组合

章Ⅱ：玩家学会用"双操作"思考
     世界 = 两种结构的张力场，攻击 = 分配律的展开/折叠

章Ⅲ：玩家学会用"扩张"思考
     世界 = 可以升维的知识空间，攻击 = 在更高域中看清结构

章Ⅳ：玩家学会用"内蕴"思考
     世界 = 空间本身是动态的，攻击 = 沿曲率梯度的流动

章Ⅴ：玩家学会用"算子"思考
     世界 = 函数的空间，无穷维，攻击 = 作用于波函数

章Ⅵ：玩家成为"函子"
     世界 = 不再有"世界"这个概念
     攻击 = 态射本身，玩家与对象等价
```

## 同一概念在不同结构层次的面孔

| 概念 | 章Ⅰ 群版本 | 章Ⅱ 环版本 | 章Ⅲ 域版本 | 哲学含义 |
|---|---|---|---|---|
| **子结构** | 子群 $H \leq G$ | 理想 $I \trianglelefteq R$ | 中间域 $F \subseteq K' \subseteq K$ | 每层都有"更小的自己" |
| **商结构** | 商群 $G/H$ | 商环 $R/I$ | 商域（域无非平凡理想）| 压缩信息的艺术 |
| **同态** | 群同态 $G \to G'$ | 环同态 $R \to R'$ | 域同态（必是单射！）| 结构保持映射 = 知识的翻译 |
| **自同构全体** | $\text{Aut}(G)$ | $\text{Aut}(R)$ | $\text{Gal}(K/F)$（伽罗瓦群）| 自对称的全体 = 最深的结构信息 |
| **原子分解** | Sylow定理 | 唯一因子分解（UFD） | 多项式在分裂域的完全分解 | 找到"质因数" |
| **零/单位** | 只有单位元 $e$ | 零元 $0$ + 单位元 $1$ | 同上，但 $0$ 是"孤岛" | 从"一个零"到"零被隔离" |

## IK-FK 双向验证矩阵

| 章 | FK（数学→玩法，推导出的机制） | IK（目标体验→选择的数学） | 同构点 |
|---|---|---|---|
| Ⅰ | $D_4$ 的8个元素 = 8种攻击 | 想要"摧毁对称=伤害"| 群阶 = 攻击数量，自然成立 |
| Ⅱ | 零因子 $ab=0$ = 武器失效区 | 想要"环境陷阱有数学意义"| 零因子定义直接给出玩法 |
| Ⅲ | $V_4$ 交换群 = "打两次归零" | 想要"需要理解才能打的Boss"| 每个元素是自身的逆，优雅 |
| Ⅳ | 测地线 = 最短路径武器 | 想要"弯曲空间中的运动感"| 测地线定义即玩法 |
| Ⅴ | $\hat{H}\psi = E\psi$ = 本征值弱点 | 想要"量子感的战斗"| 薛定谔方程 = Boss 设计蓝图 |
| Ⅵ | 函子 = 玩家操作变成元操作 | 想要"玩家成为游戏本身"| 范畴论的自指 = 最终体验 |

**结论**：∂WORLD 是迄今为止 IK 和 FK 对齐度最高的游戏设计实验。  
这不是因为数学"刚好符合"，而是因为数学的结构**本来就是**人类直觉的最精确表达。
