信号参数（BW / Fc 可调）

BW = 120 Hz，T = 0.2 s（固定）
Fc = 250 Hz（教学载波，真实 LoRa 在 GHz）
通带范围 = 40 ~ 160 Hz
基带频率 = −60 ~ +60 Hz
N = BW·T = 24 chips，Δf = 1/T = 5 Hz（固定！）

当前状态

fsymbol = s × 5 Hz = 0 Hz
采样点数 Nsamp = 40
基带 Nyquist = Fs/2 = 100 Hz
信号最高基带频率 = ±60 Hz

为什么符号是 0~23？

N = BW × T = 120 × 0.2 = 24 chips
每个符号对应循环移位 0~N-1 步，共 24 种可能
实际 LoRa SF7：BW=125kHz，T=1.024ms → N=128
符号越多 = 每符号承载更多 bit，但解调也更耗 CPU

混叠说明

基带信号：瞬时频率范围 −60 ~ +60 Hz
Fs=400Hz：Nyquist=200Hz > 60Hz → 无混叠
Fs=200Hz：Nyquist=100Hz > 60Hz → 也无混叠
Fs=100Hz：Nyquist=50Hz < 60Hz → 混叠！
时域图的连续线是理论公式画的，不反映折叠；stems 采样点值才是 ADC 实际采到的（已混叠）

⚠️ 为什么 FFT 必须用 24 个 chip 采样？

真实 LoRa：Fs = BW = chip rate，N_samp = N_SYM = 24
循环移位 chirp 的两段（s·Δf 和 (s-N)·Δf）在 24 点 DFT 中
因为 e^{-j2πn}=1 而完全叠加到同一个 bin s
若用 Nsamp=40（Fs=200Hz），两段在 DFT 网格上错位
→ 频谱泄漏！峰值两侧出现假相关
⑨ FFT 图已修复为始终用 24 chip 采样计算

①.① 采样后等效离散频率（折叠图）
ADC 采样后，任何超出 Nyquist 的分量都会折叠（aliasing）回 [−Fs/2, Fs/2]。 公式：f_alias = f − round(f/Fs)·Fs。 200Hz 时基带±60Hz 完全在 Nyquist 100Hz 以内，折叠图和原图相同 → 无混叠。 切换到 100Hz 采样，±60Hz 超出 Nyquist 50Hz，可以看到分量折叠跳变！
r(t) 折叠频率 c*(t) 折叠频率 乘积折叠频率 Nyquist 限
② 接收信号 r(t)：通带参考波形
射频侧"真实"波形 Re{rBB(t)·ej2πFct}，40→160 Hz 扫频。这是模拟信号，DSP 看不到。
③ 接收信号 r(t)：基带 I/Q ★（DSP 真正看到的）
硬件 IQ 下变频 + ADC 采样得到复数序列 r[n] = I[n] + jQ[n]。圆点 = ADC 采样点（带噪声）。混叠时：虚线=真实连续信号，彩色实线=干净 sinc 重建（显示混叠形状）。SNR<∞：灰色实线=带噪声 sinc 重建，展示噪声如何改变重建波形（sinc 匹加权平均将部分噪声抖平滑）。
I（连续参考） Q（连续参考） I[n] Q[n]
④ 共轭 down-chirp c*(t)：通带参考波形
瞬时频率 160→40 Hz（高到低），与 up-chirp 相反。模拟信号，DSP 看不到。
⑤ 共轭 down-chirp c*(t)：基带 I/Q ★
本地预存复数序列 c*[n]。I 与 up-chirp 相同，Q 取负号 = 数学共轭（相位取反）。c* 带宽同样是 ±BW/2；混叠时：虚线=精确 c*(t)，实线=sinc 重建（欠采样后混叠版），这才是乘法实际相乘的对象。
I = cos(φup) Q = −sin(φup) I[n] Q[n]
⑥ 乘积 r(t)·c*(t)：通带参考波形
连续域乘积，频率不再变化 → 通带纯余弦。连续参考，DSP 见不到。
⑦ 乘积 y[n] = r·c*：基带 I/Q ★（FFT 的输入）
s≠0 时实际是两段不同频率（s·Δf 和 (s−N)·Δf）——这就是为什么必须用 N_SYM=24 个 chip 采样做 DFT：两段在 24 点网格上共振叠加 = 单峰。混叠时：虚线=真实连续信号，彩色实线=干净 sinc 重建。SNR<∞：灰色实线=带噪声重建，显示噪声如何扭曲 FFT 前的输入信号。
I = Re{y[n]} Q = Im{y[n]} I[n] Q[n]
⑧ 极坐标复平面图（相量轨迹）
采样点在复平面（I=横轴，Q=纵轴）上的轨迹。 r[n] 和 c*[n] 都在单位圆上变速旋转（chirp = 角速度在变）。 y[n]=r·c* 的轨迹是匹速旋转（正弦！）——这里展示的是过采样的 prdSamp（可视化用）。 时间顺序：浅色→深色。
接收 r[n]（变速chirp）
共轭 c*[n]（反向变速chirp）
乘积 y[n]（匀速旋转！）
⑨ FFT 幅度谱 |Y[k]|（解调结果）
对 y[n] 做 N_SYM=24 点 DFT（chip-rate 采样，t_n = n·T/24），与真实 LoRa 一致。 关键：N_samp 必须 = N_SYM = 24，否则循环 chirp 的两段无法消去相位跳变 → 频谱泄漏（两侧相关性假性升高）。 Fs=200/400Hz 只是时域可视化用的过采样，DFT 始终用 chip rate 的 24 点。 峰值 bin = 符号 s，低 SNR 时误码。
峰值 bin（检测到的符号） 其他 bins 真实符号 s
📐 离散域具体运算步骤（DSP 实际做的事）
Step 0 ▸ 硬件 IQ 下变频 + ADC（不是软件做的） 天线 s_RF(t) = Re{r_BB(t)·e^{j2πF_c·t}} 混频器1：s_RF × cos(2πF_c·t) → LPF → I_analog(t) 混频器2：s_RF × (−sin(2πF_c·t)) → LPF → Q_analog(t) 两路 ADC → r[n] = I[n] + j·Q[n] n = 0,1,…,N_samp−1 Step 1 ▸ 本地预生成共轭 down-chirp（存表，不实时计算） φ_up[n] = 2π·(−BW/2·n/Fs + BW/(2T)·(n/Fs)²) c*[n] = cos(φ_up[n]) − j·sin(φ_up[n]) ← Q 取负 = 数学共轭 Step 2 ▸ 逐点复数乘法（de-chirp 核心，仅 N_samp 次复乘） r[n] = a + j·b c*[n] = c − j·d (d = sin(φ_up[n])) y[n] = r[n]·c*[n] = (ac+bd) + j·(bc−ad) ← 相位相减 Step 3 ▸ DFT（N = BW·T 点，对应 N 个符号） Y[k] = Σ y[n]·e^{−j2πkn/N} k = 0,…,N−1 峰值 bin k* → 解出符号 ŝ = k* （N_samp 可 ≠ N，pad/resample 后对齐） Q 路为什么不能省略？（负频率与解析信号） cos(φ) 是实偶函数：cos(+φ) = cos(−φ)，频谱双边对称 → 无法区分 e^{+jφ}（上扫）和 e^{−jφ}（下扫） I+jQ = e^{jφ} 是解析信号，频谱只有单边（无负频率混叠） → DFT 后 k=5（上扫）和 k=N-5（下扫）分开，不会撞在一起 关于 IQ 硬件（真实 LoRa 芯片） RF = 868 MHz，BW = 125 kHz → 基带 ±62.5 kHz 芯片内部：模拟 IQ 混频（868MHz × cos/sin）→ 模拟 LPF → ADC ADC 采样率 = ~1 Msps（远超 Nyquist 125kHz），然后数字抽取降至 BW 最终基带复数采样率 = BW = 125 kHz，DSP 处理 N = 2^SF 个点/符号
