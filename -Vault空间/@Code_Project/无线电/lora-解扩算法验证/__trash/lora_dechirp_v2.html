📡 LoRa De-chirp 参数

固定参数

BW = 120 Hz，T = 0.2 s
Fc = 100 Hz（载波）
通带范围 = 40 ~ 160 Hz
基带瞬时频率 = −60 ~ +60 Hz
N = BW·T = 24 chips，Δf = 5 Hz

当前状态

fsymbol = s × 5 Hz = 0 Hz
采样点数 Nsamp = 40
基带 Nyquist = Fs/2 = 100 Hz
混叠：是（|f|>Nyq 部分会折叠）

⚠️ 关于通带波形的诚实说明

图②④⑥ 的"连续通带波形"是用 2400 点画的模拟参考，不是真正以 Fs 采样的离散信号。
真正的接收机只有基带 I/Q 采样点（图③⑤⑦），DSP 全程在基带域运算，从来不处理通带信号。
通带图仅供你看清"扫频"是怎么回事；要看混叠对实际算法的影响，请看 ③⑤⑦。

■ 橙 共轭 down-chirp c*(t)
■ 绿 乘积 r·c*

实线 = 连续参考波形，圆点 = 实际采样点
① 时频图（基带瞬时频率）
蓝线斜向上（接收 up-chirp，s>0 时绕回）；橙线斜向下（共轭 down-chirp）； 两者瞬时频率相加 = 绿色水平直线 = 单一频率正弦。 虚线 = ±Fs/2（Nyquist）， 超出虚线的频率分量在采样后会折叠到内部 → 混叠！
r(t) 瞬时频率 c*(t) 瞬时频率 乘积瞬时频率 Nyquist 限
② 接收信号 r(t)：通带参考波形
射频侧"真实"波形 Re{rBB(t)·ej2πFct}，瞬时频率 40→160 Hz 扫频。这是模拟信号，DSP 看不到。
③ 接收信号 r(t)：基带 I/Q ★（DSP 真正看到的）
硬件正交下变频后通过两个 ADC 同时采样 I 和 Q。每个时刻得到一个复数 r[n] = I[n] + jQ[n]。圆点之间的线只是视觉辅助，DSP 只有圆点。
I = cos(φr+φ₀)（连续参考） Q = sin(φr+φ₀)（连续参考） I[n] Q[n]
④ 共轭 down-chirp c*(t)：通带参考波形
由本地 up-chirp 取共轭得到。瞬时频率 160→40 Hz（高到低），与 up-chirp 相反 — 这就是"共轭"在时域的物理含义。
⑤ 共轭 down-chirp c*(t)：基带 I/Q ★
本地预存的复数序列 c*[n]。注意 I 与 up-chirp 相同，Q 取负号 → 这就是数学上的共轭。
I = cos(φup) Q = −sin(φup) I[n] Q[n]
⑥ 乘积 r(t)·c*(t)：通带参考波形
连续域上两个扫频信号相乘，频率不再变化 → 通带为 Fc+s·Δf 的纯余弦。注意：此图为连续参考，DSP 见不到此信号。
⑦ 乘积 r·c* ：基带 I/Q ★（FFT 的输入）
DSP 真正算出的复数序列 y[n] = r[n]·c*[n]。理想情况下 I/Q 都是频率 = s·Δf 的纯正弦。FFT 一打就找到符号 s。这是 LoRa 解调的核心。
I = Re{y[n]} Q = Im{y[n]} I[n] Q[n]
📐 离散域具体运算步骤（DSP 实际做的事）
下面是 LoRa 接收机 de-chirp 的真实流程，全部在采样点层面操作。
Step 0 ▸ 硬件 IQ 下变频 + ADC（不是软件做的） 天线信号 s_RF(t) = Re{r_BB(t)·e^{j2π Fc t}} 混频器 1： s_RF(t) · cos(2π Fc t) → LPF → I_analog(t) 混频器 2： s_RF(t) · (−sin(2π Fc t)) → LPF → Q_analog(t) 两路 ADC 同时采样 → r[n] = I[n] + j·Q[n] n = 0,1,…,Nsamp−1 Step 1 ▸ 本地预生成共轭 down-chirp（一次性，存表） φup[n] = 2π·(−BW/2 · n/Fs + BW/(2T) · (n/Fs)²) c*[n] = cos(φup[n]) − j·sin(φup[n]) ← 注意 Q 取负 Step 2 ▸ 逐点复数乘法（de-chirp 的核心，仅 Nsamp 次复乘） 对每个 n： r[n] = a + j·b (a = I_r[n], b = Q_r[n]) c*[n] = c + j·d (c = I_c[n], d = −sin(φup[n])) y[n] = r[n]·c*[n] = (a·c − b·d) + j·(a·d + b·c) Step 3 ▸ FFT (Nsamp 点) Y[k] = FFT{ y[n] }， k = 0,…,Nsamp−1 峰值 bin k* 对应频率 k*·Fs/Nsamp，即 解出符号 ŝ = k*。 — 例（s=3, Nsamp=40）— y[0] = (0.999 + j·0.044)·(1.000 − j·0.000) ≈ 0.999 + j·0.044 y[1] = ( ... )·( ... ) ≈ 0.985 + j·0.171 → 相位增量 ≈ 2π·15/200 = s·Δf/Fs·2π y[2] ≈ 0.940 + j·0.342 y[3] ≈ 0.866 + j·0.500 ⋮ → 相位线性增长 → 单频复正弦 → FFT 峰值 bin = 3 = s ✓
