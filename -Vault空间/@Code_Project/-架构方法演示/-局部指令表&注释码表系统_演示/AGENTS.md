# 局部指令表 — 码表注释系统演示项目


## 📌 本项目专属规则

1. **遇到 `>v<` 注释**：先查 `-📇索引码表.md`，确认含义后再修改
2. **新增 `>v<` 标注前**：在码表中登记（否则维护脚本会报"未登记"）
3. **推荐库**：这里可以写如 "本项目 AHRS 算法用 `nalgebra`，禁止替换成 `cgmath`"

## 🔖 码表维护规则

运行维护检查（伪代码）：

```bash
grep -r ">v<" ./src | awk '{print emoji+主标题}' | 比对 -📇索引码表.md
  → 未登记条目 → 加入码表
  → 码表有但代码无 → 删除码表条目
```

## 🗂 子目录专属规则示例

| 目录 | 专属规则 |
|---|---|
| `src/core/` | AHRS 核心算法，禁止在此引入 IO 操作 |
| `src/sensors/` | 所有传感器读取必须经 `calibration.rs` 校正 |
| `src/io/` | 环形缓冲区统一用 `RingBuffer<T>` 接口 |
