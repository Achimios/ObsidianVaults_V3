▍拉氏变换 → 矩阵方程

P₁(s) = m₁s² + b₁s + (k₁+k₂) P₂(s) = m₂s² + b₂s + (k₂+k₃) Δ(s) = P₁(s)·P₂(s) − k₂² ← 4阶，4个极点 [P₁(s), −k₂ ] [X₁] [F₁] [−k₂, P₂(s)] [X₂] = [F₂] H₁₁(s) = X₁/F₁|F₂=0 = P₂(s)/Δ(s) 零点：P₂(s)=0 H₂₁(s) = X₂/F₁|F₂=0 = k₂/Δ(s) 无有限零点（分子=k₂） H₂₂(s) = X₂/F₂|F₁=0 = P₁(s)/Δ(s) 零点：P₁(s)=0 H₁₂(s) = X₁/F₂|F₁=0 = k₂/Δ(s) = H₂₁(s) ← Maxwell-Betti 互易

▍零点的物理含义（动力减振器）

H₁₁ 零点 ≈ ±j√((k₂+k₃)/m₂) 是 m₂ 子系统的固有频率。
当激励 ω = √((k₂+k₃)/m₂) 时：m₂ 共振，吸走所有能量，m₁ 完全静止。
调节 k₂, k₃, m₂ 使零点对准干扰频率 → 主动调谐动力减振器。
🎵 两个固有模态 + 模态频率分裂

▍无阻尼时的两个模态

ωn₁ = √((k₁+k₂)/m₁) — m₁ 单独固有频率 ωn₂ = √((k₂+k₃)/m₂) — m₂ 单独固有频率 耦合后产生两个模态（b₁=b₂=0，极点在虚轴上）： Mode 1 (同相)：ω_I < min(ωn₁, ωn₂) — 两块同向运动 Mode 2 (反相)：ω_II > max(ωn₁, ωn₂) — 两块反向运动 k₂ 越大 → 耦合越强 → 两峰分得越开 k₂ → 0 → 弱耦合 → 两峰合并到各自 ωn

▍实验：把 b₁=b₂=0 看极点位置

无阻尼时 4 个极点全在虚轴 → Bode 出现两个无穷大峰 → 极点图热图在虚轴两处爆炸。
逐渐增大 b → 极点左移进 LHP → 峰值有限 → 系统渐近稳定。

▍动力减振器实验步骤

1. 设 k₁=1, m₁=1 → ωn₁=√2≈1.41 rad/s
2. 设激励 ω = 1.41 rad/s（正好在 m₁ 共振点）
3. 调 k₂, k₃, m₂ 使 √(k₂+k₃)/m₂ = 1.41 → H₁₁ 零点打在激励频率上
4. 观察 x₁(t) → 稳态幅值趋近 0 （m₂ 替代 m₁ 振动）
🔄 Maxwell-Betti 互易定理

▍定理陈述

对被动线性系统：H₁₂(s) = H₂₁(s) 即"力在 m₂ 观测 x₁"= "力在 m₁ 观测 x₂"，传递函数完全相同。

▍为什么

Cramers 法则：H₂₁ = 余子式[1,2]/Δ = k₂/Δ H₁₂ = 余子式[2,1]/Δ = k₂/Δ 余子式[1,2] = 余子式[2,1] ← 矩阵对称（对称质量-弹簧-阻尼系统）
能量角度：在频率 ω 下，从 A 传到 B 的功率 = 从 B 传到 A 的功率（被动守恒）。
