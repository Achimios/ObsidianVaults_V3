## 核心数学直觉（对话推导，备查）

### ROC 与解析延拓
- `F(s) = ∫f(t)e^{-st}dt`，σ=Re(s) 是收敛因子，σ 越大越能压制 f(t)
- ROC 永远是"最右极点实部的右侧"，极点在 ROC 左边界上
- 多极点：最右极点遮住了其他极点对积分的影响；更左的极点通过有理函数的解析延拓才"看到"
- 解析延拓 = 把公式用到积分失效的区域；不改变形状，唯一确定（类比等比数列 1/(1-x) 延拓到 |x|>1）

### 时域是卷积，s 域是乘法
- `y(t) = u(t) ★ h(t) = ∫u(τ)h(t-τ)dτ`（不是相乘！）
- 卷积定理：`Y(s) = U(s)·H(s)`（s 域才是逐点乘法）
- H(s) 双重身份：① 算子（映射 U→Y）；② 冲激响应 h(t) 的拉氏变换

### 为什么有一个正实部就发散
- Y(s) 部分分式后：`y(t) = Σ Aᵢ·e^{pᵢt} + Σ Bⱼ·e^{sⱼt}`
- t→∞ 时，实部最大的那项主宰一切
- 系统极点 Re(p)>0：h(t) 自身增长，不需要输入就自爆
- 输入极点 Re(s_f)>0：u(t) 越来越大，拖着 y(t) 增长（即使 h 在衰减）

### RHP 极点的物理限制
- 被动弹簧-阻尼系统不可能有 RHP 极点（能量守恒）
- 需要正反馈延迟 或 主动电源 才能产生 RHP 极点

### 双质量块传递函数零点规律
系统：Wall─k1─m1─k2─m2─k3─Wall

| 输入→输出 | 类型 | 零点 |
|---|---|---|
| F1→X1（自驱动） | 对角 | m2 子系统固有频率 |
| F2→X2（自驱动） | 对角 | m1 子系统固有频率 |
| F1→X2（互易） | 非对角 | 无有限零点（分子=k2，常数） |
| F2→X1（互易） | 非对角 | 同上（Maxwell-Betti 互易定理） |

- 自驱动有零点 = 另一质量块可在该频率共振吸走能量（动力减振器原理）
- 非对角无有限零点 = 只要 k2≠0，耦合弹簧总会传递能量

**性能分析**
- h(τ) 预计算：一次性对系统做冲激 RK4，存储 T=40s 窗口，约 1600 点
- 实时卷积：每步 O(W/DT)=O(1600) 次乘加 → 约 3200 ops/step
- speedP=0.01 时：~0.6步/秒 → 极流畅，慢放每帧都能看清楚被积函数在移动
- speedP=0.5 时：~30步/秒 → 4.8万 ops/sec，JS 毫无压力
- **结论：不卡，比差分迭代开销差不多**

**卷积窗口长度选择**
- h(t) 半衰期 = 0.693/σ，99% 衰减 = 5/σ
- 默认参数 σ≈0.17 → T99≈29s，取 40s 窗口足够
- 用户可以通过"冲激响应窗口"滑块调节
