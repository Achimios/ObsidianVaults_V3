🎯 激励、H(s) 曲面、稳态与暂态的统一视角

▍传递函数 H(s) 是一个复平面上的曲面

对二阶系统 mẍ + bẋ + kx = F(t)，对两边做 Laplace 变换（假设零初始条件）：
(ms² + bs + k) X(s) = F(s)
所以 H(s) = X(s)/F(s) = 1/(ms² + bs + k) = (1/m) / [(s−p₁)(s−p₂)]
在复平面 (σ, ω) 上，|H(σ+jω)| 是一个三维曲面。在极点处 (s=p₁ 或 p₂) 值趋向无穷大，形成"峡谷峰顶"。

幅频响应曲线 = 沿虚轴的切面：令 s = jω（纯虚数），则：
H(jω) = 1/(mj²ω² + bjω + k) = 1/((k−mω²) + j(bω))
|H(jω)| = 1 / √((k−mω²)² + (bω)²)
这正是右侧幅频响应图上的曲线——极点越靠近虚轴（ζ越小），虚轴附近的曲面越陡，共振峰越高。

▍Laplace 全解：稳态 + 暂态 统一在一个式子里

当有激励 F(t) = F₀·cos(ωt) 且初始条件不为零时，全 Laplace 解为：
X(s) = H(s)·F(s) + [(ms + b)·x₀ + m·ẋ₀] / [ms² + bs + k] ↑ ↑ 受迫响应（决定稳态） 初始条件响应（决定暂态）
两项的分母完全相同，共享同一对极点 p₁, p₂。对两项分别做逆 Laplace 变换：
  • 稳态部分 x_p(t) = F₀·|H(jω)|·cos(ωt + ∠H(jω))   ← 由激励频率和 H(jω) 决定
  • 暂态部分 x_h(t) = A·e−σt·cos(ω_d·t + ψ)      ← 由极点位置决定衰减速率和振荡频率

初始条件 (x₀, ẋ₀) 只影响暂态部分的幅值 A 和初相 ψ，不改变衰减速率 σ=b/(2m) 和振荡频率 ω_d。

▍极点位置决定暂态"指纹"

极点位置σ (衰减率)ω_d (振荡频率)暂态行为稳定性
左半平面 Re(p) < 0σ = −Re(p) > 0|Im(p)| > 0 (欠阻尼)指数衰减振荡渐近稳定 ✓

极点图的几何直觉：改变 b (阻尼) 时，极点沿半径 ωn 的圆弧转动，而非直线平移。
当 b 从 0 增大到临界阻尼 b_c = 2√(km) 时，极点从虚轴（纯振荡）沿圆弧转到负实轴（临界衰减）。 改变 k 或 m 时，圆弧半径 ωn = √(k/m) 伸缩，极点离原点的距离随之变化。

▍为什么 |p| = ωn = √(k/m) 恒成立？

极点：p = −b/(2m) ± j·ω_d，其中 ω_d = √(k/m − b²/(4m²))
|p|² = [b/(2m)]² + ω_d²
= b²/(4m²) + k/m − b²/(4m²)
= k/m = ωn²
∴ |p| = ωn —— 与阻尼 b 无关，只由弹簧刚度 k 和质量 m 决定
几何意义：无论阻尼多大，两个极点始终在 s 平面上以 ωn 为半径的圆弧上滑动。这条圆弧在极点图上就是那个虚线圆。
📐 系统响应数学解释（含质量 m）

▍一般解 = 瞬态响应 + 稳态响应

x(t) = x_h(t) + x_p(t)

x_h(t) = e−σt·[C₁cos(ω_d·t) + C₂sin(ω_d·t)]，σ=b/(2m)，ω_d=√(k/m−σ²)
     → 瞬态/自由响应，由初始条件 x₀, ẋ₀ 决定系数 C₁, C₂

x_p(t) = A_ss·cos(ωt−φ)，A_ss = F₀ / √((k−mω²)²+(bω)²)
     → 稳态/强迫响应，由外界激励决定

▍术语对应

外界激励 F₀cos(ωt) → 稳态响应 = 强迫响应 = x_p（同一概念，振幅由 H(jω) 决定）
初始条件 x₀, ẋ₀ → 瞬态响应 = 自由响应 = x_h（仅决定 C₁, C₂，不改变衰减速率 σ）
总响应 = 瞬态 + 稳态（线性叠加，仅适用于线性系统）

▍瞬态消逝（指数渐近）

b > 0：x_h 以指数 e−σt 衰减，时间常数 τ = m/σ = 2m/b。 理论上永远不精确到达稳态，类比电容充电渐近趋近 Vbat，5τ 后误差约 0.7%。

b = 0：x_h 不衰减，ω≠ωn 时总响应为两频率叠加，出现拍频（beat），有界但不收敛—— 相轨迹不封闭，Lyapunov 稳定但非渐近稳定。ω=ωn 时振幅线性增长，发散。

▍何时 x₀≠0 与 ẋ₀≠0「等效」？何时不等效？

F₀=0（自由振动）：两者均产生 A_h·e−σt·cos(ω_d t−ψ) 形式， (x₀, ẋ₀) 只决定振幅 A_h 和初相 ψ，稳定性行为完全相同——等效（只是相位不同）。

F₀>0（强迫振动）：稳态 x_p 固定，不同 (x₀, ẋ₀) 给 x_h 施加不同的初始「偏置」，C₁, C₂ 各不相同。 虽然最终都收敛到 x_p，但瞬态过程（振幅、持续时长、相轨迹起点）因初始条件各异——此时不等效，不能说「等效于某相位」。

▍三个频率，含义各不相同（工程常混淆！）

符号名称公式物理意义
ωn = √(k/m)无阻尼固有频率（自然频率）√(k/m) 弹簧/质量决定的"固有节拍"；无阻尼无激励时的振荡频率。= 极点模长，不随 b 改变。
ω_d = √(ωn²−σ²)有阻尼固有频率（衰减振荡频率）√(k/m−b²/(4m²)) 断开激励后实际观察到的振荡频率。= 极点虚部 Im(p)；b=0 时退化为 ωn。
ωr = ωn√(1−2ζ²)共振频率（幅频峰值频率）√(k/m−b²/(2m²)) 外力以此频率驱动时稳态振幅最大。≠ ω_d，比 ω_d 还低！ ζ>1/√2 时消失。
💡 大小关系：ωr < ω_d < ωn（b=0 时三者相等）
⚠️ 极点虚部 = ω_d（有阻尼固有频率），不是 ωr（共振频率）！极点虚部决定自由衰减振荡速度；ωr 是幅频曲线峰值位置，二者不同。
🔄 增大阻尼时，极点沿半径 ωn 的圆弧顺时针旋转（不是平移）：从虚轴（b=0，纯振荡）转向负实轴（临界阻尼，纯衰减）。

▍V 和 V̇ 的物理含义

V = ½kx² + ½mẋ² = 系统总机械能（弹性势能 + 动能）。相平面的同心椭圆 = 等能量线。
m 越大 → 相同速度下动能越大 → ẋ 轴方向的椭圆更"胖"（半轴 ∝ 1/√m）。

V̇ = kx·ẋ + mẋ·ẍ = ẋ·(kx + mẍ) = ẋ·(−bẋ + F) = −b·ẋ² + F·ẋ
注意：m 在 V̇ 中消去了！V̇ 的符号与 m 无关，只取决于阻尼 b 和外力 F。
  • V̇ < 0：净失能，轨迹向内穿越椭圆（朝原点靠拢）
  • V̇ > 0：净得能，轨迹向外穿越椭圆（远离原点）
