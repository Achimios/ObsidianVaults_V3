📐 系统响应数学解释

▍一般解 = 瞬态响应 + 稳态响应

x(t) = x_h(t) + x_p(t)

x_h(t) = e−b/2·t·[C₁cos(ω_d·t) + C₂sin(ω_d·t)]，ω_d = √(1−b²/4)
     → 瞬态/自由响应，由初始条件 x₀, ẋ₀ 决定系数 C₁, C₂

x_p(t) = A_ss·cos(ωt−φ)，A_ss = F₀/√((1−ω²)²+(bω)²)
     → 稳态/强迫响应，由外界激励的频率和幅值决定

▍三个频率，含义各不相同（工程常混淆！）

符号名称数值（本系统ω₀=1）物理意义与极点关系
ω₀ = √k无阻尼固有频率（自然频率）本演示 √k rad/s 弹簧刚度与质量之比决定的"固有节拍"——如果没有阻尼也没有外力，系统会永远以这个频率振荡。
极点模长 |s| = ω₀ 恒成立：极点坐标是 (−b/2, ±ω_d)，模长 = √((b/2)²+ω_d²) = √(b²/4 + k − b²/4) = √k = ω₀。
几何直觉：不管阻尼 b 多大，两个极点始终在 s 平面上以 ω₀ 为半径的圆弧上滑动——阻尼增大时极点沿圆弧从虚轴向负实轴靠拢，但绝不会"跑出"这个圆。
ω_d = √(k − b²/4)有阻尼固有频率（衰减振荡频率）√(k − b²/4) ≤ ω₀ 断开激励后，实际观测到的振荡快慢就是 ω_d——阻尼"压低"了振荡节奏。
= 极点虚部 Im(s)：b=0 时退化为 ω₀；b 很大时 ω_d→0，系统从"振荡衰减"变成"单调衰减"（过阻尼）。
直觉：想象弹簧悬挂重物，松开后在水里（高阻尼）vs 在空气中（低阻尼）——水中振荡明显更慢。
ω_r = √(k − b²/2)共振频率（幅频响应峰值频率）√(k − b²/2) ≤ ω_d 外力以这个频率驱动时，稳态振幅最大——是工程设计中最危险的激励频率。
注意：ω_r 比 ω_d 还低，比 ω₀ 更低！三者不相等，不能混用。
b ≥ √(2k) 时 ω_r 消失（无共振峰，高度阻尼），幅频响应从 ω=0 就开始单调下降。
💡 三者大小关系：ω_r < ω_d < ω₀（阻尼越大，差距越大；b=0 时三者完全相等）
类比音乐：ω₀ 是乐器的"理论音高"，ω_d 是加了弱音器后的"实际音高"，ω_r 是"最容易被震得嗡嗡响的频率"。

⚠️ 极点虚部 = ω_d（有阻尼固有频率），不是 ω_r（共振频率）！
极点虚部决定的是自由衰减振荡的速度；共振频率 ω_r 是另一回事——它是幅频曲线的峰值位置，比 ω_d 还低。

🔄 增大阻尼时，极点在旋转，不是线性左移：
实部 σ = −b/2 确实线性左移，虚部 ω_d = √(k−b²/4) 同步减小，但模长始终 = ω₀ = √k。
合成效果：极点沿半径 ω₀ 的圆弧顺时针旋转——从虚轴（纯振荡，b=0）转向负实轴（纯衰减，临界/过阻尼），而不是横向平移。

▍共振峰值推导（工程危险地带）

A_ss(ω) = F₀ / √((1−ω²)² + (bω)²)　← 幅频响应曲线

令 dA/dω = 0 → ω_r = √(1 − b²/2)　（b < √2 ≈ 1.414 时有解）
A_max = F₀ / (b · √(1 − b²/4))　← 共振峰幅值
b→0：A_max → ∞，无阻尼共振时振幅随时间线性增长（非指数），无稳态可言。
b ≥ √2：A_ss 关于 ω 单调递减，无共振峰，激励频率越高响应越弱。

工程案例 — 塔科马海峡大桥垮塌（1940）：风激励频率约 0.2 Hz 逼近桥身扭转固有频率， 结构气动阻尼极低（等效 b≪0.1），A_max 极高，振幅按 t 增长直至结构失效。 此后桥梁设计强制要求气动弹性分析，通过合理断面形状和附加阻尼使 ω_r 远离常见风激励区间。

▍特征方程与极点（s 平面基础）

特征方程：s² + b·s + 1 = 0
极点：s₁,₂ = −b/2 ± j·ω_d
模长 |s| = ω₀ = 1　实部 σ = −b/2 = λ　虚部 = ω_d　阻尼比 ζ = b/2
极点位置决定系统行为（极点图右侧待实现可视化）：
• 左半平面（b>0）→ 指数衰减 → 渐近稳定
• 虚轴（b=0）→ 等幅振荡 → Lyapunov 稳定（非渐近）
• 右半平面（b<0，负阻尼）→ 指数发散 → 不稳定

工程案例 — 等效负阻尼：航空发动机叶片颤振、输电线风激励舞动（Aeolian vibration）、 麦克风对扬声器的声学啸叫——正反馈使等效阻尼变负，极点越过虚轴进入右半平面。 控制工程的核心目标之一正是通过反馈将极点推入左半平面，并保持足够的稳定裕度（Gain Margin / Phase Margin）。

▍瞬态消逝（指数渐近与工程时间尺度）

b > 0：x_h 以时间常数 τ=2/b 衰减（5τ 后误差 <0.7%），类比 RC 充电 1−e−t/RC。
理论上永远不"精确"到达稳态，但工程取 5τ 为"已稳定"。

工程意义（汽车悬架举例）：阻尼太小（欠阻尼，ζ<0.5）→ 路面冲击后车身长时间摇晃； 阻尼太大（过阻尼，ζ>1）→ 响应迟钝，操控感差。最优取 ζ≈0.6~0.7（本演示 b≈1.2~1.4）， 兼顾快速收敛和抑制超调——这也是大多数工业控制器默认的阻尼比设计目标。

b = 0：x_h 不衰减，ω≠ω₀ 时两频率叠加出现拍频（beat），有界不收敛——相轨迹不封闭， Lyapunov 稳定但非渐近稳定。

▍V 和 V̇ 的相空间几何意义（李雅普诺夫第二法）

V 是什么？
V = ½kx² + ½ẋ²，正好等于系统的总机械能：前半是弹性势能（弹簧压缩/拉伸储存的能量），后半是动能（质量运动的能量）。
在相平面 (x, ẋ) 上，V = C 的曲线是椭圆等高线（右侧相图的虚线圆），每条椭圆代表一个"能量等高面"。 V 越大 = 离原点越远 = 系统蕴含的能量越多。

V̇ 是什么？
V̇ = dV/dt = kx·ẋ + ẋ·ẍ = ẋ·(kx + ẍ) = ẋ·(−bẋ + F(t)) = −bẋ² + F(t)·ẋ
这是系统的瞬时功率平衡方程：
  • −bẋ² = 阻尼耗散功率（永远 ≤ 0，能量只出不进）
  • F(t)·ẋ = 外力对系统做功的功率（可正可负，取决于力与速度方向是否相同）
V̇ < 0：阻尼耗散 > 外力输入 → 系统在净失能，轨迹向内穿越等值椭圆，朝原点靠拢
V̇ > 0：外力输入 > 耗散 → 系统在净得能，轨迹向外穿越等值椭圆，远离原点
V̇ = 0：瞬时功率平衡，轨迹沿等值椭圆切向运动（既不进也不出）

李雅普诺夫第二法的核心洞见：
• F₀=0，b>0：V̇ = −bẋ² ≤ 0 恒成立，不管轨迹在哪里，总在失能 → 轨迹只能向内穿越椭圆 → 全局渐近稳定。
  关键在于：我们完全不需要解出 x(t) 的表达式，只凭"能量函数的导数为负"这一事实就能断言稳定性。
• 有激励时（F₀>0）：稳定性由一个周期内 V̇ 的平均值 ⟨V̇⟩ 决定：
  ⟨V̇⟩ < 0 → 阻尼耗散占优，系统仍收敛到有界稳态（渐近稳定）
  ⟨V̇⟩ ≈ 0 → 激励补偿了耗散，系统在某条椭圆附近周期振荡（稳态）
  ⟨V̇⟩ > 0 → 外力持续注入能量，轨迹螺旋外扩（b=0, ω=ω₀ 的无阻尼共振即如此）

工程场景 — 电力系统暂态稳定性：发电机甩负荷后，用与动/势能类比的 Lyapunov 函数判断 系统轨迹是否始终在稳定域（某条 V 等值线）内侧，可在毫秒级给出"这次扰动会不会导致失步"的判断， 比全时域仿真快几个数量级，是大电网实时安全分析的核心数学工具。

💡 何时 x₀≠0 与 ẋ₀≠0 "等效"？何时不等效？
F₀=0（自由振动）：两者均产生 A_h·e−b/2t·cos(ω_d t−ψ) 形式，(x₀,ẋ₀) 只决定振幅 A_h 和初相 ψ，稳定性行为完全相同（等效）。
F₀>0（强迫振动）：稳态 x_p 固定，不同 (x₀,ẋ₀) 给 x_h 施加不同的初始"偏置"，C₁, C₂ 各不相同。虽然最终都收敛到 x_p，但瞬态过程（幅值、持续时长、相轨迹起点）因初始条件各不相同——此时不能说"等效于某相位"。
