# L3范数世界 V2 — PRD
> 起点：豆包生成的 L3 范数 3D 可视化 → 进化为可探索的数学世界

---

## 目标
打造一个以 **Lp 范数几何** 为主题的迷你 3D 探索游戏，体验接近低配 UE5 Editor：
可行走、有光感、有基础场景、有参数面板控制数学参数。

---

## 功能需求

### 🎮 F1 — 相机控制（最高优先级）
| 功能 | 按键 | 说明 |
|---|---|---|
| FPS 前进/后退/左/右 | W A S D | 人物视角平移 |
| FPS 上升/下降 | Space / Shift | 垂直移动 |
| 鼠标视角 | 右键拖拽 | FPS 模式下鼠标看 |
| 视角切换 | Tab | 上帝视角 ↔ 人物视角 |
| FOV 调节 | 面板滑块 | 30°–120° |
| 上帝模式旋转 | 左键拖拽 | 保留原始拖转行为 |

### 💡 F2 — 光照
- GLMeshItem shader='shaded' 模拟 Phong 光照
- 场景内一盏虚拟方向光（通过 shader uniform 注入）
- 无实时阴影（pyqtgraph 限制，标注为 Phase 2 TODO）

### 🏗 F3 — 场景对象
| 对象 | 数量 | 说明 |
|---|---|---|
| L3 球（中心） | 1 | 大球，半径 1 |
| Lp 球阵（p 可调） | 5 | 绕中心排列，p = 参数面板控制 |
| 纯平地板 | 1 | 实体 mesh，有光照 |
| L∞ 方块（p→∞） | 1 | 正方体角点 |
| L1 八面体（p=1） | 1 | 标准八面体形状 |
| L2 球（p=2） | 1 | 参考球，半透明 |
| 坐标轴 | 1 套 | X=红 Y=绿 Z=蓝，GLAxisItem |

### 🖼 F4 — 右侧 UI 面板
```
┌──────────────────────────────┐
│ 操作提示                     │
│  Tab  - 切换视角             │
│  WASD - 移动                 │
│  右键  - 转头（FPS模式）     │
├──────────────────────────────┤
│ 参数控制                     │
│  p值  [滑块 0.5 ~ 8.0]      │
│  FOV  [滑块 30 ~ 120]        │
│  移速 [滑块 0.1 ~ 5.0]       │
├──────────────────────────────┤
│ 视角模式: [上帝模式 / FPS]   │
│ [切换] 按钮                  │
└──────────────────────────────┘
```

### 📁 F5 — 代码模块化
```
beginFrom豆包_非L2范数/
│
├── main.py              ← 入口，组装窗口、启动 Qt
├── world_objects.py     ← 所有 3D 对象生成（Lp球/地板/坐标轴）
├── camera_controller.py ← FPS & 上帝视角相机逻辑
├── ui_panel.py          ← 右侧 Qt Panel
└── -PRD.md              ← 本文件
```
旧文件 `L3范数3D小游戏.py` 保留为历史版本，不删除。

## 技术约束
- Python 3.10 + pyqtgraph 0.14.0 + PyQt5 + PyOpenGL
- pyqtgraph GL 无原生阴影，Phase 2 可考虑 OpenGL raw shader
- 不引入 Pygame / Panda3D / vispy 等额外引擎
