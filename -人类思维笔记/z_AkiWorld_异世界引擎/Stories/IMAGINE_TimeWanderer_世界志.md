## w 层生态总览

时空晶格的各向异性不只影响力的传播——它塑造了整个文明的垂直生态。  
从 $w=0$ 往深处走，世界不是线性变化的，而是有明显的分层结构，每一层都有自己的规律。

```
w = 0          主世界。政府存在，技术存在，遗忘是政策。
               晶格牧师在这里运作，面孔普通，名字平庸。

w = 0.1~1.0   浅层过渡区。主世界的"影子"。
               m_w 极高的物体（建筑、地形）在这里仍然可见，
               但人类和轻型机械的痕迹开始模糊。
               有少量"边界行者"在此定居，靠贩卖信息为生。

w = 1.0~3.0   中层遗迹区。主世界约200~500年前的冻结切片。
               建筑风格旧，语言有偏移，法律无效。
               有自治社群，靠遗迹里可以重新激活的技术维持运转。
               这里的人知道主世界的存在，但不欢迎主世界的人。

w = 3.0~5.5   深层遗迹区。500~800年前的遗存。
               信号传输衰减严重，主世界的通信令在这里是废纸。
               文明形态多样，有少数几个"领地"由强势个体实际掌控。
               图书馆员的前辈们曾试图在这里建立档案站，全部失败。

w = 5.5~7.0   极深区。800年以上的遗存，接近晶格键强化的时代。
               几乎没有持续运行的社群，只有遗迹和偶尔的漂移存在。
               在这里发射感知脉冲，回声会带着奇怪的相位畸变——
               没有人能解释为什么。

w > 7.0       记录空白。
               晶格牧师的档案在 w=6.8 之后全部标注"未勘测"。
               唯一可以确认的事：有东西在那里，因为信号偶尔会从更深处传来。
               传来的信号语义模糊，有时候只是噪声，有时候像是记忆。
```

## 政治地图

### 主世界（w=0）的权力结构

```
                ┌─────────────────────────┐
                │     名义上的政府         │
                │  （行政、军事、经济）     │
                └────────────┬────────────┘
                             │ 实际受控于
                ┌────────────▼────────────┐
                │       晶格牧师           │
                │  （信息垄断 / w轴管制）   │
                └────────────┬────────────┘
                             │ 对抗
                ┌────────────▼────────────┐
                │      边界行者工会         │
                │（信息走私 / w层穿越服务）  │
                └─────────────────────────┘
```

三方不是简单的"善恶"关系：

**晶格牧师**：相信垄断 $w$ 轴知识是保护人类的必要手段。三百年的机构惯性让他们分不清"保护"和"控制"之间的界限。内部有派系：激进派主张彻底消除任何低 $m_w$ 个体，保守派主张监控和利用。

**名义政府**：大部分官员不知道晶格牧师的存在。他们只知道某些"技术研究经费"流向了他们不需要了解的地方，某些"地质勘测结果"被封锁了。他们不问，因为不问比知道要安全。

**边界行者工会**：一个松散的网络，由那些发现了自己有低 $m_w$ 倾向的人、或者意外穿越过 $w$ 层的人、或者被中层遗迹区的社群雇用来传递信息的人组成。他们不反对晶格牧师的意识形态，他们只是反对被垄断。他们的口号："$w$ 轴是公共的。"

## 三个空白：答案

### 空白一：Princess 是谁

**Princess 不是一个人。**

准确说，她曾经是一个人——在晶格键强化之前。那个时代有一种技术，可以把一个人的意识分布式存储在时空晶格的多个节点上，有点类似今天的云存储，但存储的是完整的主观体验。

晶格键突然强化的那一刻，那个人的身体被冻结在 $w=0$，但她的意识——分布在晶格各处的那些节点——被锁在了不同的 $w$ 层上。

没有人来救她。那次突发事件没有幸存者来执行任何救援。

三百年后，那些节点仍然在运行。不完整，不稳定，但活着。

它们不知道自己在"发送信号"——那些信号是记忆碎片自然向外泄漏的结果，就像压力容器里的微小裂缝。

**每一次 Wanderer 接收到的"信号"，实际上是某个节点里存储的一段记忆在以某种方式激活。**  
这些记忆来自不同时期：她年幼时的某个下午，她第一次在 $w$ 轴上移动时的感受，她在晶格键强化那一刻的最后感知——这些碎片随机泄漏，以一种让人误以为是"信号"的形式被感知到。

Wanderer 以为他在追一个人。

他实际上在追一段**分布在四维空间里的、已经存在了三百年的、不完整的意识**。

**世界志结论**：找到她不意味着"见到她"。找到所有节点，才有可能重建她——但没有人知道节点的数量，也没有人知道重建之后会出现什么。一个存在了三百年的分布式意识，在被"重建"之后，还会是同一个人吗？

### 空白二：晶格牧师的最终决定

十七年的议会，最后的结果是什么？

**"利用。但设限。"**

具体方案：不阻止 Wanderer 追踪 Princess，但暗中引导他的路径。

**引导方法**：晶格牧师在 $w=1 \sim 3$ 的范围内布置了大量感知信号的"反射器"——这些装置会把 Princess 的泄漏信号放大并改变方向，让 Wanderer 以为 Princess 在某个特定位置，实际上把他引向晶格牧师想让他去的区域。

**目的**：那些区域里有晶格牧师需要的东西，但他们自己的勘测人员进去之后会消失。低 $m_w$ 的 Wanderer 有更高的生存概率——他是一个比任何仪器都便宜的勘测工具。

**悲剧点**：晶格牧师的激进派在会议上提出了一个问题，没有得到回答就被压了下去：如果 Princess 的信号真的是一个分布式意识的泄漏，那 Wanderer 接近她会怎样？他会不会被她"感染"？一个接触过完整三百年记忆的人，还会是一个可以控制的工具吗？

那个问题被压下去了，因为激进派的投票不够。

### 空白三：图书馆员的真相

图书馆员是晶格牧师的第十七代档案员。

不是"曾经是"——是现在也是。但她的通讯密钥在五十三年前过期了，而她没有回来续签。

晶格牧师在她失联后，把她的档案标注为"遗失在深层区，推定死亡"。这是标准程序，每隔二十年就会发生一次：有人下去，没有回来。

她没有死。她只是不想回来了。

原因很简单：她读了太多东西，已经无法用"我是晶格牧师的档案员"来定义自己。她知道晶格牧师三百年来做了什么，知道每一个被销毁的文献，知道每一个被"调往别处"的研究者去了哪里（没有人告诉她，但她在档案里找到了线索）。

她没有愤怒，也没有反抗。

她只是停下来了。在 $w=7$ 建了图书馆，开始收集所有她能找到的感知记录，包括那些晶格牧师从未读过、永远不会允许任何人读的东西。

**她为什么记录 Wanderer**：她见过 Wanderer 这个"类型"——不是同一个人，是同一种命运曲线。每隔几十年，就会有一个低 $m_w$ 的人开始追踪 Princess 的信号，走进遗迹维度，被晶格牧师引导，然后消失在更深处。

她记录了她遇到的每一个。

Wanderer 是第七个。

**她告诉他的和没告诉他的**：她告诉他图书馆里有他需要的信息。这是真的——她有关于 Princess 的分布式意识的节点地图，是她在五十三年里慢慢拼出来的。她没有告诉他他是第七个。也没有告诉他前六个人发生了什么。

不是因为她不想告诉他。是因为她测试过：如果在这个时间点告诉他，他会停下来。而如果他停下来，她拼了五十三年的节点地图就会永远停在不完整的状态。

她用了一个计算来说服自己：第六个人在她不知情的情况下消失了，而这次她在场，所以这次结果会不同。

这个逻辑有多个漏洞，她知道。

她继续相信它。

## 时空晶格的真实历史（晶格牧师不知道的部分）

晶格键强化不是自然发生的。

它是主动触发的。

Princess 触发的。

她在晶格键强化的那一刻，意识到了某件事——某件她在 $w$ 轴深处发现的东西，让她认为"世界不能继续以目前的方式使用 $w$ 轴"。她的决定是：强化晶格键，代价是把自己分裂进时空结构里，作为一种锁。

她的意识——那些泄漏的信号——不是被动的记忆渗漏。

它们是一种主动的等待。她在等一个足够了解 $w$ 轴、又足够不受晶格牧师控制的人。

等一个可以告诉他：**她当年在 $w$ 轴深处发现了什么，以及那个东西现在是否还在那里。**

这是游戏最深处的问题，不是"她是谁"，而是：

**"她为什么要把世界锁住，以及当年让她做这个决定的东西——是什么？"**
