# 线性代数基础

有限维希尔伯特空间的直觉来源。$\mathbb{R}^n$ 和 $\mathbb{C}^n$ 都是希尔伯特空间，标准内积为：

$$\langle x, y \rangle = \sum_{i=1}^n \bar{x}_i y_i$$

## 向无限维推广时会遇到什么

| 有限维 | 无限维（希尔伯特） |
|---|---|
| 基底有有限个向量 | 需要可数正交基（Schauder基）|
| 所有范数等价 | 范数不再等价 |
| 子空间自动闭合 | 子空间可能不闭合 |
| 线性泛函自动有界 | 需要额外条件（Riesz定理） |
