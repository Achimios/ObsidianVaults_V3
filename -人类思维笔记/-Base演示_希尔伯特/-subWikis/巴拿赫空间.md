# 巴拿赫空间

完备的赋范向量空间。希尔伯特空间是特殊的巴拿赫空间——其范数由内积诱导。

## 与希尔伯特空间的关系

$$\text{希尔伯特空间} \subset \text{巴拿赫空间} \subset \text{赋范空间} \subset \text{度量空间}$$

判断标准：若巴拿赫空间的范数满足**平行四边形恒等式**，则它是希尔伯特空间：

$$\|u+v\|^2 + \|u-v\|^2 = 2(\|u\|^2 + \|v\|^2)$$
