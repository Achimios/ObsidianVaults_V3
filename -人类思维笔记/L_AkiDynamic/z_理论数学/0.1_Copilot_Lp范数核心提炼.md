## 一、Lp 范数分工律（数据库 + 语义系统）

| 范数 | 几何意义 | 工程用途 | 操作规则 |
|---|---|---|---|
| **L¹（曼哈顿）** | 绝对值之和 | 数据库索引、TF-IDF、B+树查询代价 | 多字段权重线性累加 |
| **L²（欧几里得）** | 内积空间 | 语义向量、余弦相似度、PCA/SVD | 所有语义操作只在此完成 |
| **L（切比雪夫）** | 取最大分量 | 精准极值筛选、最终排序命中 | 切换后只做排序，绝不回转或变换 |

**核心口诀**：L¹ 管离散存储，L² 管连续语义，L 管极值精确命中。

### L²L 切换的正确流程

```
1. 全部操作（归一化、降维、语义过滤）在 L² 完成
2. 不改向量坐标值，只切换度量规则为 L：sim = max_i |q_i - v_i|
3. 仅做排序 / Top-K 截断
4. ❌ 禁止：切换后再做线性变换、旋转、归一化、插值
```

## 二、稀疏性与 ℓₚ 范数

### 稀疏的定义
向量绝大多数分量为 0，只有极少数非零有效分量。信息高度集中，冗余极低。

### ℓₚ 族对比

| 范数 | 定义 | 特性 | 工程可用性 |
|---|---|---|---|
| **ℓ₀** | 非零元素个数 | 最严格稀疏，NP-hard | ❌ 不能直接用 |
| **ℓₚ (0<p<1)** | (Σ\|xᵢ\|ᵖ)^(1/p) | 比 ℓ₁ 更稀疏，非凸 | ✅ 离线微调 |
| **ℓ₁ (LASSO)** | Σ\|xᵢ\| | 凸优化、快速求解 | ✅ 首选稀疏工具 |

**ℓ₀ 为何 NP-hard**：找最少非零元素 = 组合选择问题，C(n,k) 维度爆炸，无快速算法。

### 工程应用
- **图像压缩**：DCT/小波变换域后 99% 系数接近 0  截断小系数，只存大系数
- **低SNR信号恢复**：噪声稠密分布，有用信号稀疏  ℓₚ 约束精准提取
- **特征选择**：ℓ₁ 正则化自动零化冗余特征

## 三、索伯列夫负范数

### 定义（工程直白版）
量化信号的"抖动、振荡、毛刺、超调、高频噪声"程度。振荡越剧烈  负范数越大。

**傅里叶域计算**：
$$\|f\|_{H^{-s}} = \left(\int |\hat{f}(\xi)|^2 \cdot (1+|\xi|^2)^{-s} \, d\xi\right)^{1/2}$$
加权低频、抑制高频  量化信号非光滑程度。

**正范数 vs 负范数**：

| | 索伯列夫正范数 H⁺ˢ | 索伯列夫负范数 H⁻ˢ |
|---|---|---|
| 量化 | 函数光滑性、导数阶数 | 振荡程度、超调、发散风险 |
| PID用途 | 评价响应平滑性、延迟 | 评价振荡/超调/洗桨冲击 |

### 嵌入式近似（STM32 可用，无需 FFT）
```python
# 简化版索伯列夫负范数：时域近似
oscillation_score = variance(gyro) + peak_to_peak(gyro) + max(overshoot)
```

### 替代传统 PID 损失函数
```
L(Kp, Ki, Kd) = ||e(t)||_{H^{-s}} + λ正则项
```
比 ISE/ITAE 更精准：关注振荡平滑性，不只看误差大小。

## 四、四旋翼无人机自动 PID 系统  工具分层部署

> 适用扩展：AkiDynamic 机械狗、轮式、双足机器人的控制层设计参考同一原则。

### 架构总览

```
第1层（离线PC）     黑匣子分析，整定基础PID + 动态规则
第2层（50Hz RPi）   实时工况识别（重载/洗桨/强风），切换脚本
第3层（200Hz~2kHz STM32） 动态PID参数调整
第4层（2kHz STM32） 角速度闭环，最低延迟，绝不发散
```

---

### 第4层：2kHz 内环（绝对实时，STM32）

**允许工具（全部 O(n) 时域）**：

| 工具 | 作用 |
|---|---|
| 线性卡尔曼 LKF | 陀螺去噪，不丢瞬态 |
| 扰动观测器 ESO/DOB | 实时估计洗桨/阵风扰动 |
| PID 局部线性化 | 核心控制算法 |
| Teager 能量算子 TEO | 亚微秒级瞬态冲击检测 |
| 李亚普诺夫简易判稳 | 在线防参数发散 |
| L1 范数硬约束 | 钳制 P/I/D 上下限 |

**绝对禁止**：任何频域变换、NN、ℓₚ 拟范数、复杂优化。

---

### 第3层：200Hz2kHz 动态PID（STM32）

| 工具 | 作用 |
|---|---|
| EKF 状态观测 | 估计油门、电压、等效载荷 |
| 索伯列夫负范数（时域近似） | 量化振荡强度 |
| 增益调度 Gain Scheduling | TPA / Dynamic D-term / I-relax |
| 二次规划 QP | 平滑 PID 参数变化，防跳变 |
| 组 L1 约束 | 参数稀疏 + 平滑过渡 |

---

### 第2层：50Hz 机载（树莓派）

| 工具 | 作用 |
|---|---|
| VMD / EMD 模态分解 | 分离洗桨、振动、噪声模态 |
| 小波包 WPT / HHT | 时频特征提取 |
| 浅层 NN | 工况分类（重载/小桨/洗桨/强风） |
| 特征值分析 | 当前模态稳定性判断 |
| L1 稀疏降维 | 降低 NN 计算量 |

---

### 第1层：离线PC（黑匣子全自动整定）

| 工具 | 作用 |
|---|---|
| 卡尔曼 RTS 平滑 | 还原真实 gyro/motor 信号 |
| HHT + 冯卡门湍流谱 | 建模洗桨与大气扰动 |
| 索伯列夫正/负范数 | 评价响应速度 + 振荡发散风险 |
| L1 / ℓₚ 稀疏优化 | 筛选最简有效 PID 参数 |
| NSGA-II 多目标优化 | 同时最小化延迟 + 超调 + 扰动敏感度 |
| 蒙特卡洛 + 波特图/奈奎斯特 | 稳定性鲁棒验证 |
| 参数灵敏度 Sobol | 找对洗桨最敏感的 PID 参数 |

**输出**：基础 PID 参数表 + 完整增益调度脚本

---

### 瞬态检测（洗桨）最优工具选型

| 场景 | 最优工具 | 原因 |
|---|---|---|
| 在线2kHz | **Teager 能量算子** | O(n)，亚微秒级 |
| 机载50Hz | **VMD/EMD** | 最准确模态分离 |
| 离线PC | **HHT + 小波包** | 最完整时频分析 |
| ❌任何场景 | 傅里叶/DCT | 全局平均，抓不住瞬态冲击 |

## 五、绝对红线（任何层级都不可违反）

```
❌ ℓ₀ 范数：NP-hard，永远不用
❌ 曲波变换：只适合二维图像，一维信号无价值
❌ 2kHz 内环跑 NN / 频域变换：直接炸实时性
❌ L 切换后做任何线性变换/旋转/归一化
❌ 不同范数空间之间混用操作（各空间的事在各空间内完成）
❌ ℓₚ (0<p<1) 在线运行：仅限离线微调
```
