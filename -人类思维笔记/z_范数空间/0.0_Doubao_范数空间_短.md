## 1. 第一核心：旋转不变性（宇宙不偏爱任何方向）

### 1.1 L2 长度公式

∥x∥2​=x2+y2+z2​

- x,y,z：三个坐标
- 上标 2：平方
- ⋅​：开平方

**物理意义：**

这是现实中尺子量出来的**真实长度**。

### 1.2 旋转不变的数学表达

把向量随便旋转一个角度，得到新向量 x′，必须满足：

∥x′∥2​=∥x∥2​

**物理意义：**

- 宇宙没有 “特殊方向”
- 向东转、向西转，**长度不变、能量不变、物理规律不变**

### 1.3 内积是旋转不变的根源

⟨u,v⟩=ux​vx​+uy​vy​+uz​vz​

- ⟨u,v⟩：内积
- ux​,vx​：x 方向分量相乘

**物理意义：**

- 内积 → 决定**角度、正交、投影**
- 旋转时：
    
    ⟨u′,v′⟩=⟨u,v⟩
- **只有 L2 对应的内积，旋转后完全不变**

---

### 对比：非 L2 为什么破坏旋转？

比如 L1：

∥x∥1​=∣x∣+∣y∣+∣z∣

拿一个简单向量旋转 45° 看：

x=(1,0,0),∥x∥1​=1

旋转 45° 到 xy 平面对角线：

x′=(22​​,22​​,0)

∥x′∥1​=2​≈1.414=1

**物理结论：**

- 一转方向，“长度” 就变
- 物理规律跟方向有关
- 力、轨道、场都会**依赖坐标系角度**
    
    → 宇宙各向异性，不可能稳定。

## 2. 第二核心：周期性 / 振荡（波、原子、轨道都靠它）

振荡、周期、波函数、原子能级、电磁波，全部来自同一个方程：

### 2.1 波动方程（全宇宙最核心方程之一）

∂t2∂2ψ​=c2Δψ

#### 逐符号解释

- ψ=ψ(x,t)：场 / 波函数（光、电子、声波、引力波）
- ∂t2∂2​：**对时间二阶导数**
    
    - 物理意义：**加速度、变化的快慢变化**
    
- c：波速（光速、声速等）
- Δ=∇2：**拉普拉斯算子**
    
    Δψ=∂x2∂2ψ​+∂y2∂2ψ​+∂z2∂2ψ​
    
    - 物理意义：**空间弯曲、聚集还是扩散**
    

#### 为什么必须是 L2？

- Δ 是**二阶线性算子**
- 对应的 “长度” 一定是 **L2**
- 解一定是：
    
    ψ=ei(k⋅x−ωt)
    
    → **正弦、余弦、周期、振荡**

### 2.2 量子力学版本（薛定谔方程）

iℏ∂t∂ψ​=−2mℏ2​Δψ+Vψ

关键还是 Δ，还是 **L2 结构**。

**物理意义：**

- 只有 L2，才有**稳定能级、轨道、周期运动**
- 电子才不会掉进原子核
- 原子、分子、物质才存在

---

### 非 L2 会发生什么？（公式上看）

如果强行换成 Lp 结构（p≠2），方程会变成：

∂t2∂2ψ​∝某种非线性的 ∣∇ψ∣p−2∇ψ

特点：

- 不是二阶线性
- 没有简单周期解
- 解一般两种命运：
    
    1. ψ→0 （坍塌、消失）
    2. ψ→∞ （爆炸、发散）
    

**物理结论：**

没有波、没有周期、没有原子、没有轨道。

## 3. 第三核心：时空局部平坦（连续性、光滑、不撕裂）

### 3.1 平直时空距离公式（狭义相对论）

ds2=−c2dt2+dx2+dy2+dz2

- ds：**时空间隔**
    
    物理意义：两个事件之间 “真实不变的距离”
- dt,dx,dy,dz：时间、空间微小变化
- 全是**平方项** → 本质还是 L2

### 3.2 弯曲时空（广义相对论）

ds2=gμν​dxμdxν

#### 逐符号解释

- gμν​：**度量张量**
    
    物理意义：时空的 “尺子”，引力就是尺子弯曲
- dxμ：4 维微小增量（t,x,y,z）
- 仍然是**二次型**
    
    → 仍然是 **L2 结构的推广**

#### 局部平坦原理

在足够小区域（微观局部）：

gμν​≈平直时空的度量

**物理意义：**

- 时空不会突然撕裂、跳跃
- 可以微分、可以光滑变化
- 保证**因果性、连续性、局域性**

---

### 非 L2 时空长啥样？（芬斯勒几何）

ds=F(x,dλdx​)dλ

- 不是平方，不是二次型
- 没有内积
- 没有自然角度
- 局部不 “光滑、简单”

**物理结论：**

- 宏观用：因果混乱、轨道不稳定
- 普朗克尺度用：可以，因为极小、蜷曲，不影响宏观

## 4. 能量守恒：公式化、只给核心

### 4.1 能量 = L2 平方（最本质）

E=∥ψ∥22​=⟨ψ,ψ⟩=∫∣ψ∣2dV

- E：能量 / 概率总量
- ∫dV：对全空间积分
- 全是**平方、内积、L2**

### 4.2 守恒律（诺特定理，一句话公式）

**连续对称性 ⇨ 守恒量**

- 时间平移不变 ⇒ 能量守恒
- 空间平移不变 ⇒ 动量守恒
- 旋转不变 ⇒ 角动量守恒

全部成立的前提：

> **结构必须是二次、线性、内积、L2**

非 L2：

- 对称性破掉
- 没有守恒律
- 能量可以凭空出现 / 消失
